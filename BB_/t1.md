## 2. 사업 구조

### 사업 개요

**블랙베리는 과거 스마트폰 → 소프트웨어·보안으로 전환한 캐나다 IT 기업**이다. 3개 세그먼트: **QNX(자동차 임베디드 OS) + Secure Communications(사이버보안) + Licensing(특허)**. 과거 스마트폰 사업 청산 후 QNX 자동차 OS + 정부·기업 사이버보안 중심.

### 사업부문별 매출 구조 (FY2026 추정)

| 사업부문 | 주요 제품 | 비중(%) | 비고 |
|---------|---------|------:|-----|
| QNX | 자동차 임베디드 OS·SDV·ADAS | \~45 | 핵심 성장 |
| Secure Communications | UEM·SecuSUITE·AtHoc 사이버보안 | \~45 | 정부·기업 |
| Licensing | 특허 라이선싱 | \~10 | 안정 |
| **합계** | | 100.0 | 매출 \~$550M |

※ FY2026 추정. Cylance(소비자 보안) 매각 후 QNX + 보안 집중. 단위: 세그먼트 비중.

### 핵심 사업: QNX 자동차 OS + 사이버보안 턴어라운드

**QNX는 자동차 임베디드 OS 글로벌 선두**. 차량 SDV(Software Defined Vehicle)·ADAS·디지털 콕핏의 안전 인증(ASIL-D) 실시간 OS. 글로벌 OEM 2.5억대+ 탑재. **사이버보안(Secure Communications)은 정부·금융 보안 통신**.

- **QNX**: 자동차 임베디드 OS. SDV·ADAS·디지털 콕핏. 로열티 + 개발 수익. 차량 전장화·자율주행 수혜
- **Secure Communications**: UEM(통합 엔드포인트) + SecuSUITE(보안 통신) + AtHoc(위기 관리). 정부·금융
- **Licensing**: 특허 라이선싱 (Certicom 암호 등)

### 비즈니스 특성

- **턴어라운드 진행 (흑전 전환)**: FY2023 대규모 손상(OP -$725M) → FY2026 흑전(OP $50M, ROE 7.1%). 구조조정 + Cylance 매각 + QNX 집중
- **QNX 자동차 OS 성장 동력**: SDV·ADAS 전환으로 차량당 SW 가치 증가. QNX 로열티 + 디자인윈 백로그 성장. 자율주행 안전 OS 수혜
- **사이버보안 정부·금융 안정**: UEM·SecuSUITE 정부·금융 보안. 안정적 구독 매출
- **1년 +186% + 밸류 부담**: $3.15→$9.0. 턴어라운드 + QNX 기대. RSI 89.3 극단 과매수 + Forward PE 44.7x 밸류 부담

### 고객 구조 & 원가 구조

- **고객 구조**: QNX — 글로벌 자동차 OEM·Tier-1. 보안 — 정부·금융·기업. 라이선싱 — 특허 사용자
- **매입/원가**: SW 개발 인건비 + R&D 중심. 구조조정으로 비용 축소 → 흑전. QNX 로열티 고마진

## 4. 성장 동력 분석

**동력 1: QNX 자동차 OS (SDV·ADAS) ⭐⭐⭐⭐ (◉)**
**QNX는 자동차 임베디드 OS 글로벌 선두**(ASIL-D 안전 인증). SDV(Software Defined Vehicle)·ADAS·자율주행 전환으로 **차량당 SW 가치 증가** + QNX 로열티 성장. 글로벌 OEM 2.5억대+ 탑재 + 디자인윈 백로그 성장. 차량 전장화·자율주행 핵심 수혜. 가장 강력한 동력.

**동력 2: 흑전 턴어라운드 (구조조정 완료) ⭐⭐⭐⭐ (◉)**
FY2023 대규모 손상 → FY2026 흑전(OP $50M, ROE 7.1%). Cylance 매각 + 구조조정으로 수익성 회복. QNX·보안 집중. 흑전 정상화 + 마진 개선 진행.

**동력 3: 사이버보안 (정부·금융 안정) ⭐⭐⭐ (◎)**
Secure Communications(UEM·SecuSUITE·AtHoc) 정부·금융·기업 보안. 안정 구독 매출 + AI 보안 수요. 정부 보안 통신 락인.

**동력 4: 라이선싱·특허 ⭐⭐ (○)**
Certicom 암호 등 특허 라이선싱. 안정 고마진 수익. 양자내성암호(PQC) 특허 잠재.

## 9. 종합 판단
