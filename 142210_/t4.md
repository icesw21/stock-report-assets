# 유니트론텍 — Micron 반도체 유통 본업 + 자율주행·AI Robotics 옵션 + 26Q1 OPM 8.4% 분기 정점 vs CB 9회 행사 시작 임박 정량 검증

## Executive Summary

유니트론텍(142210)은 1996년 설립된 KOSDAQ **Micron 등 해외 Vendor 반도체·디스플레이 유통 + 자율주행·AI Robotics 신사업** 다각화 종합 IT 기업이다. 매출 구성은 (1) **반도체 72.21%** — Micron(메모리·NAND·MCP) + Western Digital·Kioxia 등 글로벌 Vendor 국내·아시아 유통, (2) **디스플레이 27.60%** — LCD·OLED·터치 패널 유통, (3) **기타 0.19%**. 자회사 **(주)오스코** 반도체·디스플레이 유통 전문 + 자율주행 SW·HW 개발 자회사 + AI Robotics 사업화. 현재가 **8,610원 / 시총 1,787억 / PER 13.54x / PBR 1.28x / ROE 8.2% / 부채비율 234.2%**로 52주 신고가 (4,930원→8,610원, **1.75배 폭등**).

폭등 동력은 (1) **2026Q1 매출 2,203억(+27.7% YoY) + OP 184억(+84.7%) + NI 165억(+187.1%) + OPM 8.4% 분기 정점 마진** — 2025년 분기 평균 OP 81억의 **2.27배 폭증**, 유통업 정상 마진(3~5%) 대비 +70% 점프, (2) **5년 매출 +96% 성장 (3,920→7,690억) 연 CAGR 18.4%** — 견조한 안정 성장, (3) **Micron 메모리 슈퍼사이클** — HBM3E·HBM4 + LPDDR5X + SSD NAND 글로벌 수요 폭증, (4) **자율주행·AI Robotics 옵션** — 정부 국책과제 + 자체 개발 누적, 매출 인식 시 옵션 가치, (5) **남궁선 21.49% + 22.54% 안정 지배구조** + 외인 +318,193·기관 +184,567 동시 매집 + 자사주 1.25% + 배당 2.6%(DPS 165원).

리스크는 (1) **CB 9회 10.0% 인-더-머니(+38.1%) 행사 시작일 2026-05-21** — 어제부터 매물 가능, 1,925,236주 매물벽 (시총 9.3%), (2) **유상증자 결정 2025-06-27** — 자본 확충 진행 중, 추가 희석 가능성, (3) **52주 100% 상단(1.75배 폭등) + RSI 71.9 + MA200 +42.3% 과열**, (4) **부채비율 234.2%** — 유통업 평균 수준이나 추가 차입 어려움, (5) **25Q2 NI -48억 일회성** — 분기 변동성, (6) **유통업 마진 한계** — OPM 8% 천장 (Micron 단가 변동 직격), (7) **메모리 사이클 종속성** — Micron capex·D램·NAND 사이클 둔화 시 매출 직격.

확률가중 적정주가 **8,161원 (-5.2%)** — 현재가와 거의 일치하는 적정 영역. **CB 9회 행사 매물 소화 + 26Q2 OPM 6%+ 유지 + 자율주행 매출 인식** 3가지 트리거 충족 시 PBR 1.6x (+25%, 10,747원) 진입 가능. 자율주행·AI Robotics 매출 인식 시 PBR 2.0x (+56%, 13,434원) 극낙관 시나리오 가능 (확률 5%). Fwd PER 2.71x(26Q1 연환산) 기준은 매우 저평가이나 유통업 특성상 PER 8~12x 적정 영역.

**전략**: 보유자 30% 익절 + 잔량 홀드. 신규 진입 1차 **6,752원(MA20, -22%)** 30% / 2차 **6,233원(CB 행사가, -28%)** 30% / 3차 **5,889원(MA60, -32%)** 30%. SL 4,930원 (52주 저점, BPS 6,717의 -27% 마진). 1년 목표 10,000원 (+16%), 2년 12,000원 (+39%). CB 행사 매물 소화가 단기 최대 변수. 자율주행 매출 인식이 중기 강력한 옵션 가치이나 시점 미확정.
