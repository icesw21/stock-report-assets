# 에이팩트(200470) — SOCAMM2 양산 + 엔비디아 베라루빈 사이클: OCI/유니드 SI 인수와 OSAT 후공정 변신

---

## Executive Summary

에이팩트는 ① 2025-11 발표된 **SOCAMM2(LPDDR5X) 테스트 양산**으로 NVIDIA Vera Rubin(2026H2~2027H1) 차세대 AI 메모리 폼팩터에 직접 노출되고, ② 2026-06-10 **다이내믹그로우쓰(OCI/유니드 SI)에 최대주주 1,230억(주당 5,249원) 양도**로 거버넌스가 PEF Exit → SI 본업 시너지 모드로 전환되며, ③ 2026-01 **음성공장 재가동**으로 capa가 30% 이상 확대된다. 분기 OP가 -34억(25Q1)→9억(Q2)→20억(Q3)→**57억(Q4 OPM 17.6%)**으로 8분기 연속 가속됐고 매출원가율은 117.9%→91.1%로 27%p 정상화됐다. **확률가중 적정주가 11,825원(현재가 9,980원 대비 +18.5%)**, 다만 1년 +377% 랠리·CB+BW 19.6% 희석·PEF 양도단가 +90% 괴리로 단기 변동성 큼.

## 1. 분석 배경

에이팩트는 2007년 SK하이닉스 협의회 회원사 공동출자로 설립된 국내 유일 **PKG+TEST Turn-Key + MCP(Memory+SystemIC) 통합 테스트** OSAT다. 2022년 패키징 사업 양수로 일괄 공정 체제를 완성한 후 2023~24년 메모리 다운사이클로 누적 적자 -715억을 견뎠고, 2025년 매출 1,110억(+28%), OP 53억(흑전), Q4 OPM 17.6%로 본업 회복이 검증됐다. T1 성장동력 분석에서 **'SOCAMM2 양산'과 'PEF→SI 양도'가 ⭐⭐⭐ 동률**로 평가됐으나, 두 동력이 사실상 단일 사이클(엔비디아 베라루빈 + OCI/유니드의 반도체 소재 시너지)에 결합되므로 본 T4는 통합 주제로 분석한다.

T1 성장동력 분석에서 "SOCAMM2 양산 + 엔비디아 베라루빈 사이클"을 ⭐⭐⭐ 최고 중요도로 평가.

## 2. 산업/시장 개요

### SOCAMM2 폼팩터 — NVIDIA Vera Rubin AI 메모리 모듈의 표준화

**SOCAMM(Compression Attached Memory Module)**은 NVIDIA가 GB200/GB300 NVL72 플랫폼에서 LPDDR5X 기반 데이터센터 메모리를 모듈화하기 위해 도입한 폼팩터다. **SOCAMM2**는 2026~2027년 출시되는 NVIDIA Vera CPU + Rubin GPU 플랫폼(VR200 NVL144)의 표준 메모리 모듈로 채택될 예정으로, 기존 RDIMM/MRDIMM 대비 **전력 효율 +30%, 대역폭 +25%, 패키지 면적 -40%** 개선을 제공한다. 

핵심 변화는 **메모리가 GPU 보드에 직접 모듈화 장착**된다는 점이다 — 이는 메모리 ASP를 끌어올리는 동시에 패키징 단계의 새로운 가치를 창출한다(다이 → MCP → 모듈 적층). 2025-11 SK하이닉스 192GB SOCAMM2, Micron 256GB LPDDR5X SOCAMM2, 삼성전자 SOCAMM2 양산 발표가 동시 진행됐고, NVIDIA가 LPDDR5X 채택 단가를 HBM3E 대비 1/3 수준으로 낮춰 **AI 추론용 코스트 다이내믹스를 재정의**하고 있다.

### NVIDIA 베라루빈 사이클 — 2027~2030 GPU 플랫폼 변화

NVIDIA의 GPU 로드맵은 Blackwell(2024~25) → **Rubin(2026Q4 발표, 2027 양산)** → **Rubin Ultra(2027Q4~2028)** → Feynman(2029~)로 이어진다. Rubin 플랫폼은 ① CPU(Vera) + GPU(Rubin) Tier-1 통합, ② NVL144 랙 단위 표준화, ③ 메모리 측면에서 **HBM4 + LPDDR5X SOCAMM2 듀얼 메모리 아키텍처**를 채택해 추론 비용을 낮춘다.

OSAT 관점에서 사이클 구조는: **HBM4 패키징은 CoWoS·SoIC 등 첨단 패키징 영역(TSMC 직접) → SOCAMM2/LPDDR 모듈 패키징·테스트는 메모리 OSAT 외주**로 분리된다. 즉, 에이팩트 같은 메모리 OSAT는 **HBM 사이클의 후방·보조 영역인 LPDDR5X SOCAMM2 모듈 후공정에서 실질 수혜**를 본다.

**SOCAMM2 진입 경로**: ① SK하이닉스 협의회 회원사 공동출자 DNA(2007 설립부터 18년 거래) → SOCAMM2 외주 1차 후보, ② 2025-11 SOCAMM2 테스트 양산 발표 = 라인 인증·검증 단계 통과 시그널, ③ **음성공장 재가동(2026.01)을 매각 추진하던 라인을 다시 가동한다는 점**에서 회사의 물량 확신이 가장 강한 신호. 4) MCP 통합 테스트 노하우(국내 유일) → SOCAMM2 모듈의 다중 다이 검증에 직접 적용.

**SI 인수 활용**: 2026-06-10 다이내믹그로우쓰(OCI/유니드) 변경 → ① OCI그룹의 반도체 가스(과산화수소·황산·질산)·실리콘 사업과 후공정 라인 결합, ② 유니드의 KCl·KOH 반도체 케미칼 + 다이내믹그로우쓰의 전략 자본 → **글로벌 OSAT 진출(중국·동남아 fab 외주)의 자본·네트워크 확보**.

### 경쟁우위 요소

- **소규모·전문 OSAT의 라인 셋업 속도**: 하나마이크론(매출 1.5조)·SFA(1.6조) 대비 1/15~1/4 규모지만 신규 폼팩터 라인 셋업·검증·양산 진입은 더 빠름. 음성공장 idle 라인 재가동만으로 capa +30% 확보.
- **MCP 통합 테스트 국내 유일**: SOCAMM2 모듈은 LPDDR5X 다이 + PMIC + 컨트롤러를 모듈 단위로 통합 검증해야 함 → 에이팩트 MCP 노하우가 직접 적용. 메모리 OSAT 진영에서 차별화.
- **SK하이닉스 DNA + 삼성 신규 채널**: 2007년 SK하이닉스 협의회 회원사 공동출자 → 18년 신뢰. 2021H2부터 삼성전자 패키징 신규 거래 → 한국 IDM 양사 동시 노출.
- **OCI/유니드 SI 결합**: 반도체 가스·실리콘·케미칼 + 후공정 패키지화 = 단순 OSAT를 넘어 **반도체 소재·서비스 통합 플랫폼**으로 리포지셔닝 잠재력.
- **음성공장 idle capa의 옵션 가치**: 매각 추진하던 라인을 재가동한다는 사실은 CAPEX 부담 최소·물량 확신·BEP 매출 낮은 ramp 모두 충족.

## 6. 밸류에이션 영향

### PER 리레이팅 논리

현재 PER 80.9x(EPS 123원, NI 52억 기준)는 **2025년 흑전 첫 해 EPS의 일회성**이며, 2026년부터는 OP가 본격 정상화되면서 EPS 기준이 점프한다. 

PER 밴드 비교: 하나마이크론 44.8x, SFA반도체 15.2x, Powertech 12~18x, ChipMOS 10~15x. 에이팩트는 **소형 OSAT 프리미엄 + SOCAMM2 신규 폼팩터 옵션 가치**로 35~60x 구간이 정당화될 수 있으나, 양산 검증 전까지는 **기본 시나리오 PER 45x**가 적정선. 양산 풀 ramp 가시화 시 60x로 리레이팅, 실패 시 25x로 디레이팅 가능.

EPS 추정 (희석 19.6% 반영, 발행주식수 42.4M → 50.7M):
- 2026E 보수: NI 60억 / 50.7M = **120원** / 2026E 기본: NI 130억 / 50.7M = **256원** / 2026E 낙관: NI 200억 / 50.7M = **394원**

## 9. 결론

에이팩트의 T4 핵심 메시지는 **"국내 유일 MCP Turn-Key OSAT가 NVIDIA Vera Rubin SOCAMM2 사이클의 1차 수혜와 OCI/유니드 SI 결합이라는 두 개의 트리거를 동시에 보유했고, 그 기대가 1년 +377%로 이미 절반 이상 반영됐다"**는 것이다. 확률가중 적정주가 11,775원은 현재가 9,980원 대비 +18.0% 업사이드로, **양산 검증과 SI 시너지가 분기 실적으로 확인되기 전까지는 추격매수보다 PRZ 8,117~8,267원 분할 진입이 합리적**이다. CB+BW 19.6% 희석은 점진적이지만 가시적 부담이며, PEF 양도단가 5,249원과 시장가 +90% 괴리는 신규 대주주의 차익 매도 인센티브로 작용할 수 있다. 다만 음성공장 재가동(매각 추진하던 라인)과 SOCAMM2 테스트 양산 발표는 회사가 받은 물량 확신이 가장 강한 신호로, **2026Q2~Q3 분기보의 패키징 매출 YoY 가속 + OPM 두 자릿수 정착이 확인되면 낙관 시나리오로 리레이팅 가능**하다.

**모니터링 체크포인트**:
- 2026Q1~Q2 분기보 패키징 매출 YoY +50% 지속, 음성공장 재가동 가동률 공시
- 2026-06-10 최대주주 다이내믹그로우쓰 변경 + 후속 IR/사업협력 공시 (OCI 가스·유니드 케미칼 결합 패키지)
- 2026-07-18 CB 행사 개시 후 분기보 신주발행 추이 (월 단위 희석 속도)
- NVIDIA GTC 2026 Spring/Fall 키노트 — Rubin/Rubin Ultra SOCAMM2 채택 일정 공식 확인
- SK하이닉스/삼성전자 SOCAMM2 양산 ramp 분기 매출 (TrendForce/디일렉 보도)

- 글로벌 OSAT 점유율 변화 (한국 4.3% → 5%+ 도달 여부) + 에이팩트 글로벌 진출 뉴스 (중국·동남아 fab 외주)
