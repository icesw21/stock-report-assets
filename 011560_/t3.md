## 6. 밸류에이션 논평

현재 23,900원은 **PER 7.08x + Fwd PER 4.1x + PBR 0.86x + 순현금 시총 42%** 극단 저평가. 확률가중 적정주가 **32,204원 (+34.7%)**.

**프리미엄/디스카운트 요인**:

| 요인 | 방향 | 조정폭 |
|---|---|---:|
| 26Q1 매출 +40.9% + OP 158억 정점 갱신 | ↑ | +25%p |
| Fwd PER 4.1x + PBR 0.86x 극단 저평가 | ↑ | +20%p |
| 순현금 +1,042억 (시총의 42%) | ↑ | +20%p |
| CB/BW·유증·스옵 0건 + 김우영 47.47% 안정 | ↑ | +15%p |
| 반도체 EPC 슈퍼사이클 (삼성·SK·한화) | ↑ | +15%p |
| ROE 13.1% Peer 수준 | ↑ | +10%p |
| 매출 사이즈 감소 (10,320 → 7,158억) | ↓ | -10%p |
| 부채비율 168% (운영부채 중심) | ↓ | -5%p |
| 단기 MA200 +54% 일부 과열 | ↓ | -5%p |

**재평가 트리거**:
1. 26Q2~Q4 분기 매출 2,000억+/OP 100억+ 안정
2. PBR 1.0x 도달 (27,833원, +16%) — 기본
3. PBR 1.5x 도달 (41,750원, +75%) — 낙관
4. 자사주 매입 또는 배당 확대 공시
5. 반도체 EPC 대형 신규 수주

**결론**: **명백한 가치투자 영역 + 26Q1 추세 확정 + 순현금 시총 42%** = 강력 매수 시그널. 확률가중 +34.7% 업사이드. **즉시 분할 매수 또는 MA20 (22,322원) 영역 분할 매수 권장**. PBR 1.5x 도달 시 41,750원 (+75%), PER 6x 연환산 시 34,782원 (+46%) 가능.
