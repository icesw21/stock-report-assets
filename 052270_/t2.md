# 아이윈플러스(052270) 기술적 분석

⚠️ **데이터 결손**: KIS 시세 데이터 미수신 — 차트·MA·RSI·MACD·BB 등 모든 기술 지표 산출 불가.

## 1. 가격 현황

- 현재가/52주 고저/거래량: **데이터 결손**
- DART 직접 조회 또는 증권사 HTS 확인 필요

## 2. 분석 한계

- 차트 패턴 분석 불가
- 보조 지표 산출 불가
- 기술적 시그널 종합 불가

## 3. 권장 사항

⚠️ **거래·상장 상태 직접 확인 필수**:
1. DART 공시 직접 조회 (관리종목·거래정지·상장폐지 여부)
2. 증권사 HTS에서 거래 가능 여부 확인
3. (주)우전 사명 변경 이력 확인
4. 거래량·매물대 분석은 펀더멘털 검증 후 진행

기술적 분석은 거래 정상화 후 재작성 권장.
