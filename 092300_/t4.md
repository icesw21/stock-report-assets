# 현우산업 — 자동차 전장·OLED PCB 슈퍼사이클과 25Q3 OPM 7.9% 정점·PBR 0.72x 가치 재발견 정량 검증

---

## Executive Summary

현우산업 주가 +68% 폭등 (2,630→4,420원, 6개월) + 2026-04 박스권 (3,000~3,500원) 돌파 + 5,000원 일시 정점 후 4,420원 조정. 1987년 설립 PCB 제조 전문기업 (1996 법인전환·KOSDAQ 상장). 핵심 동력은 (1) **25Q3 매출 643억(+28.7% YoY) + OP 51억(OPM 7.9%) 사이클 정점 갱신** — 2023 정점 (OPM 5.9%) 초과, (2) **자동차 전장·EV·ADAS PCB 슈퍼사이클** + **OLED·micro LED 디스플레이 신규 라인 수혜**, (3) **PBR 0.72x + 순현금 +215억 (이자부 64억 < 현금 279억) + FCF +219억** 청산가치 영역 + 풍부한 자금, (4) **외인 +59.1만주 (20일 강력 매집)** + **문병선 29.01% + 가족 32.19% 안정 지배**. PCB 98.3% 본업 + 내수 53% + 수출 47% 균형. 다만 (1) **CB 1건 (행사가 4,545원·희석 12.61%·잔액 120억·만기 2027-08)** 현재가 +3% 근접 = 즉시 매물 압박 임박, (2) **단기 RSI 74.7 + MA200 +36% 과열**, (3) **기관 -14만주 이탈**. 확률가중 PBR 기반 **적정주가 5,792원 (현재 4,420원 대비 +31.0% 업사이드)**. PBR 1.0x (6,129원, +39%) ~ 글로벌 평균 1.5x (9,194원, +108%) 가능. MA20 (3,834원) 영역 평균회귀 분할 매수 권장.

## 1. 분석 배경

현우산업은 2025-11 2,630원 저점에서 2026-05-19 4,420원까지 6개월간 **+68% 폭등** + 4월 박스권 돌파 + 5,000원 일시 정점. 핵심 동력:

### 동력 트리거 분석

**25Q3 사이클 정점 갱신**:
- 매출 643억 (+28.7% YoY) 분기 최대
- OP 51억 (OPM 7.9%) — 2023 정점 (OPM 5.9%) 초과
- 8분기 박스권 (분기 매출 450~564억) 돌파

**자동차 전장·OLED 슈퍼사이클**:
- EV·HEV 차량당 PCB 사용량 +200% (ICE 대비)
- ADAS·자율주행 PCB 채택 가속
- OLED·micro LED 디스플레이 라인 신규 증설

**가치 재발견**:
- PBR 0.72x = 청산가치 영역
- 순현금 +215억 + FCF +219억
- 외인 +59만주 강력 매집

### 분석 범위

- 자동차 전장·OLED PCB 슈퍼사이클 TAM
- 25Q3 OPM 7.9% 정점의 추세 검증
- PBR 0.72x 가치 재발견 시나리오
- CB 행사가 4,545원 매물 압박
- 사이즈 디스카운트 vs 글로벌 Peer 격차

### 한계

- FnGuide 정식 커버리지 부재
- 글로벌 PCB 단가 압박 (대만·중국)
- CB 행사 시점 매물 압박 위험
- 자동차·전장 CAPEX 사이클 의존

## 9. 결론

**핵심 메시지**:
1. **사업 본질**: 1987년 설립 PCB 전문. 자동차 전장 + LCD·LED·OLED + 가전 다각화. 내수 53% + 수출 47%
2. **사이클**: 2023 정점 → 2024 둔화 → 2025 회복 V자 + 25Q3 OPM 7.9% 사이클 정점 갱신
3. **자본구조**: PBR 0.72x + 순현금 +215억 + FCF +219억 + 문병선 32% 안정 = 가치 재발견 영역
4. **위험**: CB 행사가 4,545원 근접 + RSI 74.7 과열 + 기관 -14만주 이탈

**투자 논리**: 펀더멘털 + 가치 재발견 진행. 확률가중 +31% 업사이드. **MA20 (3,834원) 영역 평균회귀 분할 매수 권장**. CB 행사 완료 + 사이클 가속 시 PBR 1.0~1.5x (6,129~9,194원, +39~108%) 가능.

**리레이팅 조건**:
- ✅ 26Q1~Q4 분기 매출 600억+/OPM 5%+ 안정
