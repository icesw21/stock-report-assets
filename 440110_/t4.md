# 파두 T4 Deep Dive — AI 데이터센터 SSD 컨트롤러 호황 + 하이퍼스케일러 4\~5사 확보 + Gen6 진입 (1Q26 OP 77억 흑전 검증, Marvell·Microchip 시장 점유 흡수)

---

## Executive Summary

파두(440110)는 **국내 유일 + 글로벌 4대 독립 엔터프라이즈 SSD 컨트롤러 팹리스**(Marvell, Microchip, 실리콘모션, 파두). 서울대 메모리 연구실 R&D 10년 + 2015 법인 설립 + 2023 매출 절벽 사태로 시장 신뢰 충격 → **2026Q1 OP 77억 흑전 + 매출 595억(+210% YoY) V자 폭증**으로 펀더멘털 폭발 진입.

핵심 투자 논리는 **3중 결합**: ① **AI 전력 난제 + 하드웨어 가속기 차별화** — Gen5 25W/Gen6 30W+ 전력 폭증 시대에 ARM 코어 기반 Marvell·Microchip 대비 동일 전력 기준 입출력 최고 ② **북미 하이퍼스케일러 3사 확보 + 2027 4\~5사 추가** — 2027 상위 6대 클라우드 중 5사 점유 가능성 ③ **Gen6 컨트롤러 개발 완료 + 2027H2 공급 확정** — 단가 +30\~50% + OPM 25%+ 진입.

회사 2026 매출 목표 **3,000억원(+225% vs 기존 2,000억 5월 상향) + OPM 10% 중반**, 2028 매출 8,000\~9,000억 + OPM 25% 중반 가이드 공식화. 신한투자증권 BUY/TP **130,000원(+16.7%)** 신규 커버, 유안타·삼성증권 추가 커버.

리스크는 **2023 매출 절벽 재발 우려 + PBR 284.9x 극단 + 부채비율 618.5% + 1년 +1,003% 단기 차익 실현 압력**. CB 7회차 119.9억 깊은 ITM +938%(행사 시작 도래)는 희석 2.21%로 충격 제한.

**확률가중 적정주가 141,750원** (현재 111,400원 **+27.2%**) — 보수 85,000원(25%) / 기본 140,000원(45%) / 낙관 180,000원(25%) / 극단 250,000원(5%). 전략: **MA20 99,270원 분할 매수, MA60 72,482원 깊은 조정 추가 진입, SL 67,000원, 목표 140,000원**.

## 1. 분석 배경

파두 주가는 2025년 5월 10,100원 → 2026년 5월 111,400원으로 **1년 +1,003% 사상 최대 폭등**. 단순 AI 테마 동조가 아닌 ① 2023 매출 절벽 사태 졸업 ② 2026Q1 OP 77억 흑전 검증 ③ 회사 매출 목표 50% 공식 상향 (2,000→3,000억) ④ 외국인 +4.8M주 폭매수 ⑤ 신한투자증권 BUY/TP 130,000원 신규 커버 5중 결합이 만든 V자 랠리.

T1~T3에서 확인한 사실:
- 1Q26 OP 77억 흑전 + 매출 +210% V자 (5년 적자 졸업)
- 회사 가이드: 2026 3,000억 / 2027 6,000억 / 2028 8,000\~9,000억
- 신한 TP 130,000원 (PER 40.6x × EPS 3,262원)
- 북미 하이퍼스케일러 3사 + 2027 4\~5사 확보 가능성
- Gen6 컨트롤러 개발 완료, 2027H2 공급 확정

**왜 이 주제가 핵심인가?** — eSSD 컨트롤러 시장은 본질적으로 ① 검증 기간 6\~12개월 ② 신규 벤더 진입 난이도 극단 ③ 한번 진입하면 5\~7년 락인의 3가지 특성이 결합한 **승자독식 시장**. 글로벌 4사(Marvell·Microchip·실리콘모션·파두) 중 Marvell은 AI 맞춤형 칩으로 리소스 이동 + Microchip은 Gen5 성능 열위 판정 + 실리콘모션은 컨슈머 영역 집중. **파두만이 AI 데이터센터 단일 집중 + 하드웨어 가속기 차별화**로 향후 2\~3년 점유율 확장 트랙 진입.

### 분석 범위

- 글로벌 AI 데이터센터 eSSD 시장 + 전력 난제
- 하이퍼스케일러 3사 → 4\~5사 확장 시나리오
- Gen6 컨트롤러 본격 채택 효과 (2027H2\~2028)

- 회사 가이던스 2026 3,000억 / 2028 8,000\~9,000억 검증

- 확률가중 적정주가 도출 (PER/PBR/DCF 통합)

### 한계

- 회사 분기 IR 명시적 공시 빈도 낮음 → 매출 분해 추정 의존
- 하이퍼스케일러 고객사명 NDA로 미공시 → Meta 외 2사 추정

### AI 전력 난제 — eSSD 컨트롤러 차별화의 핵심

**AI 데이터센터 1대당 SSD 8\~16개 병렬 연결 → 전력 효율이 TCO 결정**:

| PCIe 세대 | 출시 | SSD 전력 (W/대) | 데이터센터 1랙 (40대) | 차별화 중요도 |
|---------|------|-----------:|---------------:|-----------:|
| Gen3 | 2017 | 8W | 320W | 낮음 |
| Gen4 | 2020 | 15W | 600W | 보통 |
| **Gen5** | **2024** | **20\~25W** | **800\~1,000W** | **높음** |
| **Gen6** | **2026\~** | **30W+** | **1,200W+** | **극단적** |
| Gen7 (예상) | 2028~ | 40W+ | 1,600W+ | 극단적 |

**파두 하드웨어 가속기 아키텍처**: 기존 업체(Marvell·Microchip)는 **ARM/RISC-V 범용 코어 추가** 방식으로 성능 향상 → 펌웨어·소프트웨어 스택 의존 + 전력 효율 한계. **파두는 반복 워크로드를 하드웨어 직접 처리** → 코어·펌웨어 비중 축소 → **동일 전력 기준 입출력 처리 성능 PCIe Gen5 시장 최고**. Gen5 마이크로칩 성능 열위 판정 + Gen6 마벨 열위 예상 (신한 5/12).

### 글로벌 eSSD 컨트롤러 공급망 변화

기존 eSSD 컨트롤러 시장은 **인하우스(IDM 자체 설계) 40% + 독립 팹리스 60%** 구조. 독립 팹리스 중:
- **Marvell**: 25% → 15%(2028E) **감소** — AI 맞춤형 칩 + 네트워킹 리소스 이동
- **Microchip**: 20% → 8%(2028E) **감소** — Gen5 성능 열위
- **실리콘모션**: 컨슈머 + 부팅 영역 집중, AI 데이터센터 진입 제한
- **파두**: 5% → **10%+(2028E)** — 위 두 회사가 빠지는 자리에 진입

**의미**: ① 글로벌 4사 중 1사만 적극적 시장 확장 진행 ② **Marvell·Microchip이 빠진 자리에 IDM(SK하이닉스·삼성·Micron)이 파두 채택** ③ 신한 시장 점유 28E 10%, 매출 1억개+/년 × $5\~7 단가 = **매출 5,000\~7,000억 가능성** ④ 추가로 키옥시아(8%)·샌디스크(4%) 등 비-인하우스 IDM 진입 가능.

## 4. 파두의 역량 분석

### 하드웨어 가속기 아키텍처 — 기술적 차별화

파두의 핵심 기술 **HW Accelerator 기반 SSD 컨트롤러**는 다음과 같이 작동:

| 항목 | 기존 (Marvell·Microchip) | **파두 HW Accelerator** |
|------|---------|--------|
| 처리 방식 | ARM/RISC-V 범용 코어 | **반복 워크로드 하드웨어 직접 처리** |
| 펌웨어 의존도 | 매우 높음 (SW 스택 필요) | **낮음 (HW에서 직접 처리)** |
| 전력 효율 | 보통 (Gen5 25W+) | **최고 (Gen5 22\~23W)** |
| 발열 특성 | 높음 | **낮음** |
| 입출력 처리 성능 | 보통 | **PCIe Gen5 시장 최고** |
| 펌웨어 개발 기간 | 6\~12개월 | **3\~6개월 (HW가 대체)** |
| ASIC 면적 | 보통 | **작음 (HW 가속)** |
| TCO 영향 | 보통 | **최적 (전력+발열+성능)** |

**핵심 강점**: 동일 전력 기준 입출력 처리 성능 시장 최고 + 발열 낮음 + ASIC 면적 작음 → **TCO 최적화 = AI 데이터센터 채택 우선 순위**.

### 진입 전략 — 낸드 IDM 협력 + 모듈하우스 + 클라우드 직납 3단 트랙

파두는 3개의 매출 트랙 병행:
1. **낸드 IDM 컨트롤러 공급** (매출 80%, 2025 740억) — 낸드 업체(SK하이닉스·삼성·Micron)가 SSD 드라이브 완성해 하이퍼스케일러에 납품. 파두는 컨트롤러 칩만 공급
2. **모듈하우스 SSD 모듈 공급** (매출 20%, 2025 184억) — 자체 컨트롤러 + 외부 낸드 결합 SSD 모듈을 Adata·Biwin에 공급
3. **클라우드 직납** (실매출 2027\~28~) — 클라우드 사업자가 직접 낸드 조달해 eSSD 제조, 파두는 컨트롤러 직납

**2026\~27 매출 폭증**: ① 낸드 IDM 매출 폭증 + ② 모듈하우스 매출 확장 + ③ 클라우드 직납 매출 인식 시작 = 3중 동력.

### 기존 사업과의 시너지

| 기존 역량 | 신사업 활용 | 시너지 효과 |
|---------|-----------|-----------|
| SSD 컨트롤러 IP | PMIC (Power Management IC) | 데이터센터 서버 전력 관리 |
| 펌웨어 + HW 가속기 | CXL (Compute Express Link) | AI 메모리 풀링 차세대 표준 |
| TSMC 협업 | 5nm/3nm 차세대 칩 | Gen6 컨트롤러 + Gen7 진입 |
| 하이퍼스케일러 영업 | 클라우드 직납 사업 | 매출 마진 확대 (낸드 IDM 통과 없음) |
| 인력 306명 (엔지니어) | 신규 R&D | 매출 +200% 성장에도 인력 +10% 충원만 |

### 경쟁우위 요소

- **하드웨어 가속기 아키텍처** — 5\~7년 후발주자가 따라잡기 매우 어려운 기술 격차
- **북미 하이퍼스케일러 3사 + 2027 5사 확보** — 글로벌 인증 + 글로벌 락인
- **TSMC 2027 웨이퍼 할당 확보** — 양산 안정성
- **Gen5 시장 최고 성능 + Gen6 개발 완료** — Marvell·Microchip 자리 흡수
- **판관비 레버리지 극대화 구조** — 매출 2배 성장에도 인력 +10% 충원만 → OPM 폭증
- **신한·유안타·삼성 메이저 3사 적극 커버** — 시장 신뢰 + 인지도 확산
- **선수금 유입 증가 + 추가 자금조달 가능성 낮음** — 유상증자 압력 없음

## 9. 결론

파두는 **국내 유일 + 글로벌 4대 독립 AI 데이터센터 SSD 컨트롤러 팹리스**로, ① 2026Q1 OP 77억 흑전 검증 ② 회사 매출 목표 2,000→3,000억 5월 상향 ③ 북미 하이퍼스케일러 3사 + 2027 5사 확보 가능성 ④ Gen6 컨트롤러 개발 완료 ⑤ Marvell·Microchip 시장 점유 흡수의 5중 결합이 12\~36개월 펀더멘털 폭증 트리거.

**Marvell·Microchip 자리 흡수 시나리오**: Marvell은 AI 맞춤형 칩으로 리소스 이동(eSSD 비중 25→15%) + Microchip은 Gen5 성능 열위(20→8%) → 파두가 5%→10%+로 시장 점유 흡수. 신한 2028 매출 8,638억 + OPM 25% + EPS 4,047원 검증 시 **PER 35\~40x = 적정주가 141,750원(+27.2%)**.
