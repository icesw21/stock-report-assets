### 핵심 사업: 엔터프라이즈 SSD 컨트롤러 글로벌 팹리스 — AI 데이터센터 TCO 최적화 1위

**서울대 메모리 연구실 R&D 10년 + 2015 법인 설립 + 2018 첫 ASIC 칩 아웃**. 컨슈머 SSD가 아닌 **엔터프라이즈 SSD(eSSD) 시장 단독 집중**. eSSD는 데이터센터·하이퍼스케일러용 고성능 SSD로, **검증 기간 6개월\~1년, 신규 벤더 진입 난이도 극단적으로 높음** — 글로벌 독립 컨트롤러 팹리스는 사실상 4사 (Marvell, Microchip, 실리콘모션, **파두**).

- **국내**: SK하이닉스 Gen5 SSD 컨트롤러 사실상 단독 공급. 삼성전자 Gen5 TLC SSD 대부분에 파두 컨트롤러 채택
- **해외**: **북미 하이퍼스케일러 3곳 확보** (Meta + 신규 2사). **2027년 상위 6개 클라우드 중 4\~5사 확보 가능성**. Lenovo, HPE 등 글로벌 서버 OEM 확장 진행
- **신사업**: ① Gen6 SSD 컨트롤러 개발 완료 (2026초 SSD 드라이브 완성, 2027H2 공급 확정 목표) ② PMIC (Power Management IC) ③ CXL (Compute Express Link) 메모리 확장 — 신한투자증권 리포트

### 비즈니스 특성

- **팹리스 ASIC 모델**: 컨트롤러 칩은 ASIC 성격으로 **세대 내 가격 변동 거의 없음** — 물량 증가가 실적 성장의 핵심 변수. Gen 세대 진화 시 단가 +30\~50% 상승 (Gen5→Gen6)
- **하드웨어 가속기 아키텍처 (차별화 핵심)**: 기존 업체(Marvell·Microchip)는 ARM/RISC-V 범용 코어 추가 방식 → 펌웨어 의존 + 전력 효율 한계. **파두는 반복 워크로드를 하드웨어 직접 처리 → 동일 전력 기준 입출력 처리 성능 PCIe Gen5 시장 최고 수준**
- **AI 전력 난제 직격 수혜**: Gen4 15W → Gen5 20\~25W → Gen6 30W+ 전력 폭증. 데이터센터 다수 SSD 병렬 연결 시 **전력 효율 차별화 = TCO(Total Cost of Ownership) 최적화 = AI 데이터센터 채택 우선 순위**
- **TSMC 2027 웨이퍼 할당 확보**: 양산 공급망 안정성 확보 — 유안타 5/26 리포트

## 4. 성장 동력 분석

**동력 1: 북미 하이퍼스케일러 4\~5사 확보 + 2027년 6대 클라우드 중 5사 점유 ⭐⭐⭐⭐⭐ (◉)**
현재 **북미 하이퍼스케일러 3사 확보 완료** (Meta + 2사 신규). 2026H2 3번째 고객 출하 시작, **2027 상위 6개 클라우드 중 4\~5사 확보 가능성** (유안타 5/26). 하이퍼스케일러 1사 추가 = 매출 +200\~400억/년 추정. **AWS·Microsoft·Google·Meta·Oracle·Tencent·Alibaba** 중 4\~5사 진입 시 매출 8,000\~9,000억원 가시성.

**동력 2: Gen6 SSD 컨트롤러 본격 채택 (2027H2\~2028) ⭐⭐⭐⭐ (◉)**
**Gen6 컨트롤러 개발 완료 + 양산 테스트 진행 중**. 2026초 Gen6 SSD 드라이브 완성 → 2027H2 공급 확정 목표. **Gen6 시장 내 속도 빠른 업체는 삼성·Micron·파두** 3사. Gen6 채택 시 컨트롤러 단가 +30\~50% 상승 + Gen5 OPM 10% 중반 → **Gen6 OPM 25%+ 진입** 가능. Blackwell 인증 완료 + Rubin 인증 마무리 단계.

**동력 3: AI 전력 난제 + 하드웨어 가속기 차별화 ⭐⭐⭐⭐ (◎)**
**Gen4 15W → Gen5 25W → Gen6 30W+ 전력 폭증**. AI 데이터센터 1대당 SSD 8\~16개 병렬 → 전력 효율 차별화 = TCO 결정 변수. **파두 하드웨어 가속기 아키텍처 = 동일 전력 기준 입출력 처리 PCIe Gen5 시장 최고**. Marvell·Microchip(ARM 코어) 대비 발열·전력·BOM 모두 우위. Gen5 마이크로칩 성능 열위 판정, Gen6 마벨 열위 예상 (신한).

**동력 4: 신사업 PMIC + CXL 메모리 확장 ⭐⭐⭐ (◎)**
신한 리포트 명시 — **PMIC(Power Management IC) + CXL(Compute Express Link)** 확장 진행. PMIC는 데이터센터 서버 전력 관리 핵심 칩, CXL은 AI 메모리 풀링 차세대 표준. 두 신사업 모두 **2027\~2028 본격 매출 인식** 예상. 컨트롤러 단일 의존도 분산 + 멀티플 리레이팅 트리거.

**동력 5: 외국인 +4,815,235주 폭매수 + 1년 +1,003% 모멘텀 ⭐⭐ (○)**
외국인 20일 +4.8M주 폭매수(시총의 8.6%) — 사상 최대 매수 규모. 기관 -2.19M 매도 vs 외국인 압도적 매수로 V자 랠리. 1년 +1,003% 폭등(10,100→111,400)으로 시장 신뢰 완전 회복. 다만 RSI 63 + 신고가 근접 +RSI 75 도달 시 단기 조정 가능.

## 8. Bull vs Bear

### 🐂 Bull Case — "AI SSD 컨트롤러 호황 + 하이퍼스케일러 확장 + Gen6 진입"

◉ **2026Q1 OP 77억 흑전 검증 + 매출 +210% 폭증** — 5년 적자 졸업, 회사 가이드 2026 3,000억(+225%) + OPM 10% 중반 트랙 진입
◉ **북미 하이퍼스케일러 3사 + 2027 4\~5사 확보 가능성** — Meta + 2사 + 2026H2 3사 + 2027 추가 진입 = 매출 8,000\~9,000억 가시성
◎ **Gen6 컨트롤러 개발 완료 + 2027H2 공급 확정** — 단가 +30\~50% + OPM 25%+ 진입. Marvell·Microchip 열위로 시장 점유 5%→10% 확장
◎ **AI 전력 난제 직격 수혜 + 하드웨어 가속기 차별화** — Gen6 30W+ 전력 폭증 시 파두 아키텍처 결정적 우위
○ **신한 TP 130,000원 + 신사업 PMIC/CXL 확장** — 멀티플 리레이팅 진행 중, 회사 가이드 상향 시 150\~180k 여지

### 🐻 Bear Case — "2023 매출 절벽 재발 우려 + PBR 285x + CB 매물"

◉ **PBR 284.9x + 부채비율 618.5%** — 적자 누적, 흑전 검증 1분기뿐. 2026Q2~Q3 실적 부진 시 -30\~40% 조정 가능
◉ **2023 매출 절벽 사태 재발 우려** — 메모리 사이클 둔화 + 고객사 도입 연기 시 분기 매출 -30\~50% 변동 위험
◎ **CB 7회차 119.9억 깊은 ITM +938% (행사 시작 도래)** — 단기 매물 출회 가능. 다만 희석 2.21%로 충격 작음
◎ **하이퍼스케일러 단일 고객 집중** — 매출 80% 낸드 IDM + 하이퍼스케일러 3사 → 1사 이탈 시 -25\~30% 매출 충격
