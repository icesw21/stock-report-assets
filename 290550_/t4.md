# 디케이티(290550) — 휴머노이드 로봇 충전 모듈 진출: BH 그룹 후공정 첨병의 트리플 변곡점 (휴머노이드 + IT OLED + 전장 WPC)

---

## Executive Summary

디케이티는 BH 그룹의 SMT/FPCA 후공정 첨병이자, 26F 하반기 북미 휴머노이드 로봇용 충전 모듈 양산 진입을 앞둔 **세 개의 J-커브가 동시에 발현되는 희소 종목**이다. (1) 27F 전장 매출 2,663억(+95% YoY) — WPC 26~30 5개년 수주 2.85조 + Auto OLED 26~28 3개년 3,300억으로 가시성 최상위, (2) 27F IT OLED FPCA 698억(+188% YoY) — 애플 26F 1모델 → 28F 4모델 ASP 1.5~2배 믹스 개선, (3) 휴머노이드 충전 모듈 — 글로벌 휴머노이드 TAM 24년 32억달러 → 30F 150억달러(CAGR 39~73%) 진입의 첫 한국 부품 베타. 확률가중 적정주가 **38,250원** (현재가 31,200원 대비 **+22.6% 업사이드**) — 보수 25% / 기본 50% / 낙관 25%. CB 만기(26-06-22) 7.08% 희석과 1년 +397% 후 단기 차익 압력은 단기 -20~30% 조정 가능성으로 작용하나, 2026 H2 휴머노이드 첫 양산 공시는 PER 26.92x → 35~40x 리레이팅 트리거.

## 1. 분석 배경

23년 매출 -22.9% / 24년 +43.9% / 25년 +5.6%의 변동성을 거친 디케이티는 25년 OP -6.5%의 **체질 개선 비용** 구간을 통과 중이다. 25Q1 OP 25억(-68%) 충격은 갤럭시 S26 출시 이연 + ESS·IT OLED·로봇 선행 R&D 비용 + 조지아·DKT MEXICO 신규 법인 셋업비가 동시에 반영된 결과로, 25년 capex 111억(-48% YoY)·재고 +49%·PPE +33% 동시 증가에서 확인된다. 27F 컨센 OP 467~510억(+100~119%)은 IT OLED와 Auto의 매출 점프 + 제품 믹스 개선을 반영한 수치이며, **이 위에 27F 추정에 미반영된 휴머노이드 옵션 가치**가 얹혀 있다. 1년 +397% 상승은 시장이 휴머노이드 옵션을 일부 반영했다는 신호이나, BH 그룹 시너지(BH FPCB → DKT SMT → BH EVS WPC) 수직계열화 + 26F 하반기 양산 공시 가시화 + 멀티 OEM 전개 가능성은 아직 컨센서스 추정에 미반영. T4의 목표는 휴머노이드 진출의 **TAM·시점·리레이팅·차별화** 4축을 정량화하고, 이를 트리플 J-커브(휴머노이드 + IT OLED + 전장 WPC)와 결합한 확률가중 적정주가를 도출하는 것이다.

### 왜 이 주제인가

T1 성장동력 분석에서 **「휴머노이드 로봇용 충전 모듈」**을 ⭐⭐⭐ 최고 중요도로 평가. 선정 근거:
- **TAM(40)**: 글로벌 휴머노이드 시장 24년 $1.55~3.2B → 30F $15~150B (CAGR 39~73%, IDTechEx/Mordor/Grand View 조합 추정), Tesla Optimus 단독 30F 100만대 양산 목표 + Figure BotQ 12,000대/년(→100,000대) + Apptronik(Google·Mercedes 후원) $5B 밸류 — 부품 사이클 본격화 직전.
- **시점(30)**: 26F 상반기 샘플 → 26F 하반기 양산 매출 인식. 5개 증권사(하나·키움·DS·리딩·ARC) 27F 추정 모두 **로봇 매출 미반영** — 첫 공시가 곧 컨센 상향 트리거.
- **리레이팅 임팩트(20)**: 현재 PER 26.92x (TTM) / 24.5x (26E) / 13.9x (27E) → 휴머노이드 부품 글로벌 평균 30~50배. 양산 가시화 시 27E 기준 PER 25~30배 적용 가능 → 적정주가 56,000~67,500원/주 상단 시나리오. PER 13.9x → 30x = 약 +116% 리레이팅 여지.
- **차별화(10)**: 디케이티 자체 설계 무선충전 모듈 + **충전시간 1/3 단축 기술** + WPC + BMS + FPCA 통합 솔루션 — 한국 부품사 중 휴머노이드 충전 토털 솔루션 보유사 유일. BH EVS(글로벌 차량용 WPC 점유율 ~30%) 그룹 시너지로 진입장벽 강화.

### 분석 범위

- 글로벌 휴머노이드 로봇 산업 동향 및 충전·전력 인프라 부품 시장

- CB 만기(26-06-22)·단일고객(SDV)·과열 신호(RSI 73.5) 등 단기 리스크 매트릭스

### 한계

- **27F 매출 컨센 미반영 옵션 가치 추정**: 5개 증권사 모두 27F 로봇 매출 = 0원으로 모델링. 따라서 본 보고서의 휴머노이드 매출 시나리오는 사측 가이던스(26F 하반기 양산) + 글로벌 동종 부품사 매출 인식 패턴 기반 자체 추정.

## 2. 산업/시장 개요

### 휴머노이드 로봇 산업 현황

휴머노이드 로봇 산업은 26F를 **개념 검증(PoC) → 파일럿 양산** 변곡점으로 통과 중이다. Tesla는 26F Fremont 1라인 100만대/년 목표 + 27F 4백만대/년 + 장기 Giga Texas 1,000만대/년 라인 계획 발표(MX·MS 단종 후 휴머노이드 라인 전환). Figure AI는 24년 7.5억달러 시리즈 B(Microsoft·NVIDIA·OpenAI·Bezos·Intel 후원) + Austin BotQ 12,000대/년 라인(→100,000대 확장) 가동. Apptronik은 26F 2월 Google·Mercedes-Benz·B Capital·QIA로부터 5.2억달러 라운드($5B 밸류) 조달, Apollo 휴머노이드 NASA 페디그리 기반 공장 협업 시작. 1X·Sanctuary·Boston Dynamics·중국 Unitree·Xpeng도 26~27F 양산 라인 셋업.

핵심 트렌드는 **(1) 양산 비용 곡선 진입** — Tesla Optimus Gen 3 목표 단가 $20K (현재 $50~100K), (2) **AI/Foundation Model 기반 단일 플랫폼 전환** — Physical AI 패러다임으로 멀티 OEM이 동일 부품 표준 채택 가능성 부각, (3) **충전·전력 인프라 부품 표준화 미정** — 무선충전 + BMS + FPCA 통합 솔루션 보유 한국·일본 부품사가 초기 벤더 진입에 유리. 시장 컨센서스(Mordor·Grand View)는 30F TAM $15~30B, 더 적극적 추정(Yahoo Finance·Smart Analytics Global·Goldman)은 30F $50~100B+, IDTechEx의 TAM 5억대 시나리오는 35F $300B+ 시사. 한국 시장 관점에서는 **중국산 부품 배제(IRA·CHIPS)** + **K-부품 직진입** 정책 윈도우가 26~28F에 집중되어 있어 디케이티의 26F 진입 타이밍이 정책적 호재와 일치.

## 4. 디케이티의 역량 분석

### 진입 전략

**3단계 침투**: (1) **샘플 단계 (26F 1H)** — 자체 설계 무선충전 모듈을 북미 B사(Apptronik / Figure / 1X 추정)에 샘플 납품. 충전시간 1/3 단축이 핵심 차별화 — 휴머노이드 가동률·ROI에 직결. (2) **양산 단계 (26F 2H)** — 첫 단일판매·공급계약 공시 → 매출 부문별 신설. 베트남 신규 라인 + 26F 조지아 공장 셋업으로 북미 OEM 현지 대응. (3) **확장 단계 (27F~)** — BH EVS WPC 글로벌 30% 점유 인프라 + BCU(배터리 컨트롤 유닛) 추가 제품 → 멀티 OEM 전개. BH 그룹 사업확장 트랙 기록(차량용 무선충전 → 레이다·히팅·BMS·EV·로봇 충전)이 동일 패턴 재현 가능성을 시사 (BH EVS 김진용 대표 CES 2026 발언).

### 경쟁우위 요소

- **충전시간 1/3 단축 기술 (자체 IP)**: 디케이티 자체 설계 — 휴머노이드 가동률 직결. 8시간 가동·2시간 충전 → 8시간 가동·40분 충전 시 ROI 27% 개선 추정. 단순 부품 공급이 아닌 **시스템 성능 알파**.
- **WPC + BMS + FPCA 토털 솔루션**: 글로벌 휴머노이드 부품사 중 3개 영역을 단일 벤더로 통합 공급 가능한 사례 희소 (Foxconn·Murata도 풀스택 미보유). 표준화 미정 단계에서 **「턴키 벤더」** 포지셔닝.
- **BH 그룹 수직계열화**: BH FPCB(원판) → DKT SMT(실장·모듈) → BHFLEX VINA(임가공) → BH EVS(시스템 통합) — 단일 그룹 내 4단계 풀스택. 글로벌 시너지 + 가격 경쟁력 + R&D 공동 분담.
- **북미 현지화 인프라**: DKT USA + DKT TECH MEXICO 25년 신설 + 조지아 공장 26F 셋업 — 한국 부품사 중 북미 현지 양산 인프라 가장 빠르게 확보. Tesla(텍사스)·Figure(오스틴)·Apptronik(오스틴) 모두 텍사스/조지아 인접 지역.
- **5개 증권사 27F 추정에 미반영 옵션 가치**: 컨센서스가 휴머노이드 매출 = 0원으로 모델링 중 — 첫 공시가 곧 컨센 상향 + 멀티플 리레이팅 동시 트리거 (이중 알파).

## 9. 결론

디케이티는 **「BH 그룹 후공정 첨병이 휴머노이드·IT OLED·전장 WPC 트리플 J-커브를 동시 발현하는 한국 유일 부품 베타」**다. 27F 본업·전장만으로 OP 467~510억(+100~119% YoY) 컨센서스가 확보된 상태에서, **휴머노이드 옵션 가치는 5개 증권사 추정에 미반영** 상태로 남아 있어 첫 양산 공시(26F 하반기 예상)는 컨센 상향과 멀티플 리레이팅(PER 13.9x → 25~30x)을 동시에 촉발한다. 26F 1년은 마진 인내 구간이지만, 27F 레버리지 + 휴머노이드 매출 인식 + 애플 IT OLED 단계적 확장이 결합되면 **3~5년 시야 매출 1.1조원·OP 1,000억원 도달** 시나리오가 합리적. 단기 -20~30% 조정 시 28,000원 이하는 분할 매수 적정 구간, 6/22 CB 만기 후 수급 정상화 시 38,250원(보수 캡)~70,000원(낙관) 추세 재개 가능.

**모니터링 체크포인트**:
- ✅ **휴머노이드 PoC → 양산 전환** — 26F 2~3분기 단일판매·공급계약 공시 + 매출 부문별 「로보틱스」 신설 여부
- ✅ **북미 IT OLED 양산 공시** — 애플 태블릿 OLED 26F 출시 → DKT 26F 하반기 매출 인식 시점 + 27F 노트북 추가 모델
- ✅ **WPC 5개년 수주 2.85조 + Auto OLED 3년 3,300억 진행률** — 분기 전장 매출 350억+ 도달이 정상 궤도 시그널
- ✅ **CB 1회차 만기(26-06-22) 후 수급 안정화** — 외국인·기관 수급 정상화 + 거래량 감소
- ✅ **BH EVS WPC + DKT SMT 시너지 매출 인식** — BH 연결 매출 기여 + DKT 단독 휴머노이드 모듈 매출 분리 인식 여부
- ✅ **27F 영업이익 467~510억 컨센 충족** — 분기 OP 100억+ 4분기 연속 실현 시 PER 25~30x 안정 정착
