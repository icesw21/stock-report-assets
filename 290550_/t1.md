### 핵심 사업: 고부가 SMT 기반 FPCA 모듈 토털 솔루션

디케이티는 **FPCB 위에 MLCC·IC-Chip 등을 SMT(표면실장기술)로 모듈화한 FPCA**를 제조하는 회사다. 단순 PCB가 아니라 **실장→모듈→어셈블리** 수직계열화로 SCM 효율성과 제품 신뢰성을 동시에 확보. **삼성디스플레이 Y-OCTA 일체형 OLED FPCA 점유율 1위**로 BH(연성PCB) 그룹의 모듈 후공정 첨병 역할.

- **국내**: 삼성디스플레이 Y-OCTA OLED용 FPCA 1위 + 국내 완성차 OEM향 차량용 OLED 디스플레이 FPCA 4월 양산 개시
- **해외**: SDV(삼성디스플레이 베트남) 수출 (전체 매출의 73.3% 수출), 북미 고객사(애플 추정) 태블릿 OLED FPCA 26F 1개 모델 → 28F 4개 모델 확대
- **신사업**: ① 휴머노이드 로봇용 충전 모듈 PoC (BH EVS 연계, 26F 하반기 샘플→양산) ② IT OLED FPCA (태블릿·노트북) ③ ESS BMS (베트남·조지아 라인) ④ EV/모빌리티 무선충전(WPC) 5개년 수주잔고 2.85조원

### 비즈니스 특성

- **수출 비중 73.3%**: 결제 약 60일 USD 현금 결제 — 환율 노출 + 대형 단일 고객사 의존 (SDV)
- **BH 그룹 수직계열화**: BH(FPCB) → DKT(SMT 실장·모듈) → BHFLEX VINA(임가공) → BH EVS(전장 모듈) — 그룹 내 후공정 핵심
- **수주가 아닌 P/O 발주**: 고객사 수시 P/O 방식 — 수주잔고 미공시 구조이나 증권사 추정 기준 WPC 26~30년 5개년 약 2.85조원, Auto OLED 26~28년 3개년 약 3,300억원
- **선행 투자 사이클 진입**: 베트남 ESS 전용 라인 구축 + 미국 조지아 공장 26F 셋업 (DKT USA + DKT TECH MEXICO 25년 신규 설립) → 25~26년 수익성 일시 둔화, 27F 레버리지 본격화

### 고객 구조 & 원가 구조

- **고객 구조**: B2B. 수출 SDV(삼성디스플레이 베트남) 압도적 1위, 국내 (주)비에이치·BH EVS·BHFLEX 등 그룹사 중심. 신규 — 북미 IT OLED 고객사(애플 추정), 북미 휴머노이드 로봇 기업(B사 추정), 국내 완성차 OEM, 글로벌 배터리 셀 3사
- **매입/원가**: FPCB(BH) + 수동소자(MLCC) + IC + 외주 임가공. 매출원가율 91.2% (25년) — 박리다매 SMT EMS 구조. 25년 capex 111억원으로 24년 대비 절반(↓48%)이나 ESS·조지아 투자 본격화로 26F 재확대 전망. 25년 FCF -66억원(투자기 진입)

## 4. 성장 동력 분석

**동력 1: 휴머노이드 로봇용 충전 모듈 — BH 그룹 차세대 캐시카우 ⭐⭐⭐ (◉)**
북미 휴머노이드 로봇 기업(B사 추정 — Figure/Apptronik/1X 중 하나)과 PoC 진행 중. **26F 상반기 샘플 납품 → 하반기 양산 매출 인식** 기대. 디케이티 자체 설계 무선충전 모듈로 **충전시간 1/3 단축**. WPC + BMS + FPCA 전사 기술 집약. 휴머노이드 시장 24년 32억달러 → 30F 1,000억달러 (CAGR ~70%) — 중국 배제(SCM 탈중국)에 따른 한국 수혜 직진입. 휴머노이드 부품 기업 멀티플 PER 30~50배 부여 사례 — DKT의 26.92x → 35~40x 리레이팅 여지. **TAM(40)×시점(30)×리레이팅(20)×차별화(10) 종합 1순위**. 단순 신규 매출 추가가 아닌 **산업 내 포지셔닝 변화의 신호**(하나증권).

**동력 2: 차량용 무선충전(WPC) + Auto OLED — 전장 사업 구조적 확장 ⭐⭐⭐ (◉)**
- **WPC**: BH EVS 연계 글로벌 완성차 OEM 다수 확보. **26F~30F 5개년 수주잔고 약 2.85조원** (DKT 점유율 ~40% 가정). 26F WPC 매출 907억 → 27F 1,493억(+65%).
- **Auto OLED**: 26년 4월 국내 하이엔드 모델 양산 개시 → 4Q26 북미 전기차 기업(테슬라/리비안 추정) 신규 양산. **26F~28F 3개년 수주잔고 ~3,300억원**. 27F 매출 1,170억(+195%).
- WPC + Auto OLED 합산 27F 약 2,663억원 — **전장 사업 단일 부문이 23년 전사 매출(2,802억) 수준**. 글로벌 전장 시장 22~29F CAGR +10%. 보완 사업이 아닌 **성장의 중심축**으로 자리 잡았음.

**동력 3: IT OLED FPCA 침투 — ASP·믹스 동시 개선 ⭐⭐⭐ (◎)**
북미 A사(애플 거의 확정) **태블릿 OLED FPCA 26F 1개 모델 양산 개시 → 27F 노트북 1개 추가 → 28F 총 4개 모델**. 태블릿/노트북 FPCA는 모바일 대비 **기판 면적 + 실장 부품 수 + 회로 복잡도 모두 증가** → ASP 1.5~2배 가정. 26F 244억 → 27F 698억(+186%). 단순 매출 증가가 아닌 **제품 믹스 개선**으로 GPM 9.0% → 10.5% (DS 추정 OPM 5.1% → 6.6%) 마진 레버리지. Omdia: 27F까지 IT OLED 침투율 12% 돌파.

**동력 4: ESS BMS — 탈중국 낙수효과 ⭐⭐ (◎)**
모바일 PCM/BMS 기술력 기반 ESS BMS 솔루션. **국내 셀 3사(LGES·삼성SDI·SK On) 모두 고객사**. 베트남 ESS 전용 라인 + 26F 조지아 공장 셋업. 26F 매출 신규 발생 → 27F 309억(2026F 102억). 미국 하원 中ESS 수입 금지 법안 + 글로벌 ESS 22~29F CAGR +28% (BloombergNEF) — 한국 셀 → DKT 낙수효과. 다만 OPM 기여는 27년부터 본격화.

## 8. Bull vs Bear

### 🐂 Bull Case — "BH 그룹 + 휴머노이드 + IT OLED 트리플 점프"
