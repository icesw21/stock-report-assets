### 핵심 사업: VIA PUF 자체 개발 + 양자내성암호(PQC) 보안칩 전문기업

**국내 유일한 PUF(Physically Unclonable Function) 자체 개발 기업**. PUF는 반도체 제조공정 미세 편차로 생성되는 칩마다 다른 고유 ID — **복제 불가능한 하드웨어 보안의 핵심 기술**. ICTK의 자체 기술인 **VIA PUF**(반도체 비아홀 활용)는 ① 수동소자 → 전력 불필요 + 보안 안정성 최고 ② 외부 키 주입 없음 → 글로벌 보안 인증 표준화 가능 ③ 대만 eMemory(NeoFuse), 시놉시스 산하 Intrinsic ID 대비 차별화 기술.

- **국내**: 삼성·KT·LG U+·SK텔레콤 IoT 단말 + 신용카드사 + 5G SA 코어망 보안칩
- **해외**: **2026년 글로벌 빅테크 보안칩 80\~90% 점유율 공급 시작** (하나증권 5/19 리포트, 유럽 보안칩 업체 대체) + 일본·동남아 라이선시
- **신사업**: ① 양자내성암호(PQC) — NIST 표준화 완료(2024.08) 후 미국 정부 + 5G SA 시장 본격 진입 ② 피지컬 AI(로봇·드론) 보안칩 ③ 스테이블코인 콜드월렛 보안 ④ 양자 시뮬레이션 (양자 솔루션 수출)

### 비즈니스 특성

- **반도체 IP/팹리스 모델 (자체 칩 양산 + 라이선스 듀얼 트랙)**: 자체 PUF 칩 직접 양산(매출 90%) + PUF IP 라이선스 수익(매출 10%). 글로벌 빅테크 진입 시 로열티 비중 점진 확대 → 마진 폭증 잠재
- **레퍼런스 베이스 사업**: 보안칩은 첫 채택 후 5\~7년 모델 라이프사이클 락인. 빅테크 레퍼런스 1건 = 글로벌 확장의 핵심 키
- **고밸류 양자/AI 보안 테마**: 트럼프 행정부 20억 달러 양자산업 지원 + NIST PQC 표준화 + 5G SA + 스테이블코인 보안 = 4중 테마 결합

## 4. 성장 동력 분석

**동력 1: 글로벌 빅테크 보안칩 80\~90% 점유 공급 ⭐⭐⭐⭐⭐ (◉)**
2026년부터 **글로벌 빅테크(유럽 보안칩 업체 대체) 단말 보안칩 공급 80\~90% 점유율**. 하나증권 5/19 리포트 명시 — **세컨드 벤더가 아닌 메인 공급사**. 양자암호업체·국내 보안 솔루션 통틀어 유례 없는 수준. **빅테크 1개 레퍼런스 = 글로벌 빅테크 5\~6사 확장 가능성**. eMemory 사례(2019 TSMC 양산 후 7배 급등) 재현 시나리오.

**동력 2: 트럼프 양자산업 20억 달러 정책 + NIST PQC 표준화 ⭐⭐⭐⭐ (◉)**
2026.5 트럼프 행정부 양자산업 지원 발표 — IBM·디웨이브·리게티·아이온큐 등 9개 업체에 **20억 달러 보조금 지급 + 지분 참여**. NIST는 2024.8 양자내성암호(PQC) 표준 확정 → 미국 정부/금융권 PQC 전환 의무화 진행. **ICTK는 NIST 표준 PQC 알고리즘을 PUF 칩에 통합 + 미국 시장 직접 진출 가능** (5G SA, 데이터센터, 디지털 자산 보안). CHIPS·IRA·BEAD 사례 — 정책 산업 모두 급등.

**동력 3: 스테이블코인 + 미국 암호화폐 보안 매출 ⭐⭐⭐ (◎)**
하나증권 리포트 별도 — **스테이블코인 부상 시 ICTK 콜드월렛 보안칩 매출**. 미국 GENIUS Act(스테이블코인 규제법) 통과(2026.5) + 글로벌 디지털 자산 시장 폭증 + Coinbase·USDC 등 보안 솔루션 수요 폭증. ICTK 5/19 리포트: **"빠르면 올해 미국에서 암호화폐 보안 매출 기대"** 명시. 2027\~28년 신규 매출 카테고리.

**동력 4: 5G SA + 피지컬 AI 보안칩 ⭐⭐⭐ (◎)**
**5G SA(Standalone) 시대 본격 전환** — 코어망 분리로 단말·기지국·코어 모두 PUF 보안칩 필수. 통신 3사 + 글로벌 통신사 진입 가능. **피지컬 AI(로봇·드론) 시대 도래** — 모든 디바이스에 PUF 보안칩 의무 채택 전망. ICTK 5/19 리포트 — "피지컬 AI 시대 보안칩 대표주자 발돋움 기대" 명시. 카드·USIM 위주 기존 보안칩에서 **로봇·드론·자동차** 전반으로 매출처 다변화.

**동력 5: 양자 시뮬레이션 솔루션 수출 ⭐⭐ (○)**
하나증권 4/21 리포트 — **양자 시뮬레이션 솔루션 신규 카테고리**. 국내 양자솔루션 수출 1호 기업 후보. 글로벌 양자 R&D 시장 진입 — 신규 매출 인식 시점 2026H2\~2027.
