# 아이씨티케이 T4 Deep Dive — 글로벌 빅테크 보안칩 진입 + 트럼프 양자정책 + NIST PQC 표준화 (국내 유일 VIA PUF 자체개발 eMemory 추격 시나리오)

---

## Executive Summary

아이씨티케이(ICTK, 456010)는 **국내 유일 PUF(Physically Unclonable Function) 자체 개발 + 양자내성암호(PQC) 보안칩 전문기업**이다. 5년 연속 적자(2025 OP -89억) + 매출 변동성 큰 초기 성장 기업이지만, **2026년 글로벌 빅테크 보안칩 80\~90% 점유 공급 시작 + 트럼프 양자산업 20억 달러 정책 + NIST PQC 표준화 + 스테이블코인 보안의 4중 결합**이 12\~24개월 변곡점.

핵심 투자 논리는 **eMemory 7배 사례 추격**: 대만 eMemory가 2019년 NeoFuse + TSMC 양산 레퍼런스 확보 후 2020\~21년 매출·주가 7배 폭등 (PBR 3x→15x→60x). ICTK도 **VIA PUF + 글로벌 빅테크 공급 레퍼런스 + 양자정책 직접 수혜**가 결합해 동일 시나리오 재현 가능성. 하나증권은 BUY/TP **60,000원(+65%)** 5/27 상향 유지.

2026F 매출 232억(+480%) + OP 58억 흑전 + ROE +13.4% 폭발적 전환. 2027F 매출 404억(+74%) + OP 195억 + ROE +27.7%. PER (Fwd 27E) 22.97x — 글로벌 양자주 평균 대비 보수적.

리스크는 **5년 연속 적자 + CB 5회차 80억 깊은 ITM(+107%) + 유상증자 4건 누적 + 신고가 RSI 75 과매수**. 1년 +228% 폭등 후 단기 -30\~50% 조정 압력 상존.

**확률가중 적정주가 62,000원** (현재 36,350원 **+70.6%**) — 보수 25,000원(30%) / 기본 60,000원(45%) / 낙관 100,000원(20%) / 극단 150,000원(5%). 전략: **MA20 24,324원 ~ MA60 19,260원 분할 매수, SL 17,500원(MA120), 목표 60,000원**.

## 2. 산업/시장 개요

### 글로벌 양자컴퓨터 + PQC 시장 개요 (2024\~2030)

**양자컴퓨터의 진화 = 기존 RSA·ECC 암호 무력화 위기**. 양자컴퓨터(현재 IBM 1,121-큐비트, 구글 70-큐비트 운영)가 2030년경 실용화 단계 도달 시 현재의 모든 인터넷·금융·국방 암호 체계가 무력화된다. 이에 대비한 **양자내성암호(PQC, Post-Quantum Cryptography)** 표준화가 NIST 주도로 2016년부터 진행되어 **2024.8 최종 표준 확정** — ML-KEM (Kyber), ML-DSA (Dilithium), SLH-DSA (SPHINCS+) 3종.

**미국 정부 PQC 전환 의무**: 미국 NSA는 2030년까지 모든 정부·국방·인프라 시스템 PQC 전환 의무화. 영국·EU·일본 동일 행보. 글로벌 PQC 시장은 2024 $1.5B → 2030 $25B 추정 (CAGR +60%).

## 4. ICTK의 역량 분석

### VIA PUF 기술 — 글로벌 차별화 포인트

ICTK의 핵심 기술 **VIA PUF**는 반도체 비아홀(via, 층간 연결 통로) 형성 공정의 미세 편차를 활용:

| 항목 | 기존 SRAM PUF | NeoFuse (eMemory) | **VIA PUF (ICTK)** |
|------|---------|-------|--------|
| 작동 방식 | SRAM 초기화 값 | e-fuse 끊김/연결 | **비아홀 형성 편차** |
| 전력 소모 | 능동 (전력 필요) | 능동 | **수동 (전력 불필요)** |
| 보안 안정성 | 보통 | 양호 | **최고** (외부 키 주입 불필요) |
| 공정 호환성 | CMOS만 | CMOS만 | **모든 반도체 공정** |
| 라이선스 수익 | 낮음 | 중간 | **높음 잠재** |
| FIPS 140-3 Level 4 | 미인증 | 부분 | **인증 완료** |

**핵심 강점**: 수동소자 → 전력 불필요 + 외부 키 주입 없음 → **보안 안정성 글로벌 최고 수준** + 모든 반도체 공정에 적용 가능 → **라이선스 수익 확장성 + 글로벌 빅테크 채택 가능**.

### 진입 전략 — 빅테크 직접 공급 + 라이선스 듀얼

ICTK는 두 가지 비즈니스 모델 병행:
1. **자체 칩 직접 양산** (매출 90%, 2025 36억) — G-CHIP, A-CHIP, K-CHIP, F-CHIP, Q-CHIP — 삼성·TSMC 파운드리 위탁 생산
2. **PUF IP 라이선스** (매출 10%, 2025 4억) — 일본·동남아 라이선시 + 향후 글로벌 확장

**2026\~27 전환**: 빅테크 진입으로 ① 자체 칩 매출 폭증 ② 라이선스 매출 비중 30%+ 확장 (eMemory 30%+ 사례). **라이선스 마진은 OPM 80%+** → 전체 OPM 25%(2026E)→48%(2027E) 폭증.

### 기존 사업과의 시너지

| 기존 역량 | 신사업 활용 | 시너지 효과 |
|---------|-----------|-----------|
| VIA PUF 자체 IP | NIST PQC 통합 보안칩 | PUF+PQC 통합 솔루션 (글로벌 유일) |
| 신용카드·USIM 보안 노하우 | 빅테크 단말 보안칩 | 이중 인증 + 보안 안정성 검증 |
| 5G 보안 (이통3사) | 5G SA 코어망 보안 | 한국 통신사 → 글로벌 확장 |
| 일본·동남아 라이선스 | 미국 빅테크 라이선스 | 글로벌 라이선스 네트워크 확장 |
| 양자보안 R&D (10년+) | 양자 시뮬레이션 솔루션 | 양자 R&D 시장 진입 |

### 경쟁우위 요소

- **국내 유일 PUF 자체 개발** — 5\~10년 후발주자가 따라잡기 매우 어려운 기술 격차
- **VIA PUF의 글로벌 최고 보안 안정성** — FIPS 140-3 Level 4 인증 완료 (글로벌 빅테크 채택 가능)
- **글로벌 빅테크 80\~90% 점유 공급 레퍼런스** — 첫 빅테크 진입 후 5\~6사 확장 가능
- **NIST PQC 표준 통합 + 트럼프 양자정책 직접 수혜** — 미국 시장 진입 우대
- **eMemory 7배 사례 재현 가능성** — 2019 TSMC 양산 후 2년 PBR 3x→60x, ICTK도 동일 시나리오
- **라이선스 매출 확장성** — OPM 80%+ 마진 차원 다름
- **하나증권 단독 적극 커버리지** — 5/27 BUY/TP 60,000원 상향, 시장 인지도 확산

## 9. 결론

아이씨티케이는 **국내 유일 PUF 자체 개발 + 양자내성암호(PQC) 보안칩 전문기업**으로, ① 글로벌 빅테크 보안칩 80\~90% 점유 공급 (2026~) ② 트럼프 양자산업 20억 달러 정책 직접 수혜 ③ NIST PQC 표준화 + 미국 정부 PQC 전환 의무 ④ 스테이블코인 + 5G SA + 피지컬 AI 신규 카테고리의 4중 결합이 12\~24개월 펀더멘털 폭발 트리거.

**eMemory 7배 사례 재현 시나리오**: 2019 TSMC 양산 후 NeoFuse 매출 본격 인식 + PBR 3x→15x→60x → 주가 7배 폭등. ICTK는 ① 빅테크 양산 시작 (2026H1) ② NIST PQC 채택 본격화 (2026H2~) ③ 트럼프 양자정책 집행 (2026\~27)의 동시 결합으로 더 강한 동력 가능.

단기적으로는 1년 +228% 폭등 + RSI 75 + CB 깊은 ITM(+107%)으로 -30\~50% 조정 압력. 중장기적으로는 **2026\~27년 흑전 검증 + PBR 정상화 + 빅테크 추가 진입 + 라이선스 폭증**의 다중 동력 결합으로 **확률가중 적정주가 62,000원(+70.6%) 진입 여지**.

**핵심 트레이딩 시그널**: ① 단기 조정에서 MA20 24,324원 ~ MA60 19,260원 분할 매수 ② 2026Q2 실적(8월) 매출 50억+ + 흑전 시점 확인 후 본격 비중 확대 ③ 빅테크 추가 레퍼런스 공시는 +30\~50% 단기 모멘텀 트리거 ④ CB 5회차 전환 진행률 모니터링.
