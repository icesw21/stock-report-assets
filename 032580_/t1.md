### 핵심 사업: 메모리 반도체 Fabless (중국 동심반도체 자회사)

**1990년 설립, 1997년 코스닥 상장한 메모리 반도체 설계·판매 Fabless 전문회사**. DRAM·Flash Memory·MCP(Multi-Chip Package) 개발·판매 + PSRAM·Mobile DDR·Serial NOR Flash까지 영역 확대. **2017년 중국 동심반도체(Dosilicon)가 인수 → 최대주주 30.20%** — 중국 메모리 굴기 직접 연계.

- **국내**: 본사 영업팀 직접 매출 + 대리점. 국내 Customer 신규 메모리 수요 대응
- **해외**: 일본·중국·유럽·미주 대리점. 수출 비중 71% — 중국 동심반도체 연계 글로벌 판매망
- **신사업**: ① Consumer Electronics ② M2M/IoT ③ **Automotive(차량용 메모리)** 시장 진출 — 자동차 메모리 경쟁력 확보

### 비즈니스 특성

- **2026Q1 폭발적 턴어라운드**: 2024년 OP -45억 적자 → 2025Q4 OP +10억 흑전 → **2026Q1 OP 43억(+793.8%), NI 53억(+784.9%)** 폭발. 매출 215억(+97.5%) — 단분기 사상 최대
- **중국 동심반도체 시너지**: 모회사 동심반도체(Dosilicon)의 중국 메모리 굴기 + 생산 capa + 글로벌 판매망 연계. 중국 반도체 자립 정책 수혜
- **NOR Flash·DRAM 사이클 회복**: 메모리 다운사이클 종료 + AI·IoT·자동차 수요 + 중국 메모리 국산화로 매출 회복
- **고밸류 + 적자 TTM**: PBR 4.74x (BPS 1,438원), 적자 TTM이나 2026 흑전 진입. 1년 +701% 폭등

## 4. 성장 동력 분석

**동력 1: 2026Q1 OP 43억 폭발적 턴어라운드 ⭐⭐⭐⭐⭐ (◉)**
7분기 연속 적자 → 2025Q4 OP +10억 흑전 → **2026Q1 OP 43억(+793.8%, OPM 20.0%), NI 53억(+784.9%)** 폭발. 매출 215억(+97.5%, 사상 최대). **연환산 시 2026 OP 160억+ 가능** → PER 12.6x. 메모리 ASP 상승 + 가동률 + 중국 시너지가 결합한 가장 강력한 동력.

**동력 2: 중국 동심반도체(Dosilicon) 시너지 + 메모리 굴기 ⭐⭐⭐⭐ (◉)**
최대주주 **중국 동심반도체 30.20%** — 중국 메모리 굴기 + 생산 capa + 글로벌 판매망 연계. **중국 반도체 자립 정책(NOR Flash·DRAM 국산화)** 수혜. 동심반도체 물량 + 피델릭스 설계 기술 결합으로 매출 폭증 잠재. 미·중 반도체 갈등 속 중국 내수 메모리 수요 직접 수혜.

**동력 3: 메모리 사이클 회복 (NOR Flash·DRAM·PSRAM) ⭐⭐⭐⭐ (◎)**
2024년 메모리 다운사이클 바닥 → 2025\~26 업사이클 전환. **NOR Flash(IoT·자동차) + DRAM + PSRAM + Serial Flash** 수요 회복 + ASP 상승. AI·IoT·자동차 메모리 수요 폭증. 다운사이클 적자 → 업사이클 흑자 폭발의 전형적 사이클 종목.

**동력 4: Automotive·IoT 신시장 진출 ⭐⭐⭐ (◎)**
Consumer Electronics + M2M/IoT + **Automotive(차량용 메모리)** 진출. 차량용 메모리는 고신뢰성 + 고마진 + 장기 공급. 자동차 전장화 + IoT 디바이스 폭증으로 NOR Flash·PSRAM 수요 구조적 성장. 신시장 진입이 중장기 동력.

**동력 5: 외국인 +477,231주 매수 + 1년 +701% 모멘텀 ⭐⭐ (○)**
외국인 20일 +477,231주 매수(시총의 1.4%). 1년 +701% (850→8,460 고가, 현재 6,810) 폭등 — KOSDAQ 상위 모멘텀. 턴어라운드 폭발 + 중국 테마 + 메모리 사이클 결합. 다만 RSI 71.6 + 변동성 극단(BB 폭 243.6%)으로 단기 조정 위험.

**밸류에이션 판단**: 증권사 커버리지 부재(소형 사이클·테마주). PBR 4.74x는 적자 BPS 착시 — **2026Q1 OP 43억 연환산 시 2026 OP 160억 → EPS 540원 → PER 12.6x**. 메모리 업사이클 + 중국 시너지 검증 시 PER 15\~20x 가능. 다만 1년 +701% 폭등 + 변동성 극단으로 단기 진입 신중. 자체 추정 적정주가 6,500\~9,000원.

## 8. Bull vs Bear

### 🐂 Bull Case — "2026Q1 OP 폭발 + 중국 동심반도체 + 메모리 사이클"

◉ **2026Q1 OP 43억(+793.8%, OPM 20.0%)** — 7분기 적자 → 흑자 폭발. 연환산 2026 OP 160억+ → PER 12.6x
◉ **중국 동심반도체 30.20% 시너지 + 메모리 굴기** — 중국 NOR Flash·DRAM 국산화 + 생산 capa + 글로벌 판매망
◎ **메모리 사이클 회복 (NOR Flash·DRAM·PSRAM)** — 다운사이클 바닥 → 업사이클 ASP 상승. AI·IoT·자동차 수요
◎ **Automotive·IoT 신시장** — 차량용 메모리 고신뢰성·고마진 진출
○ **외국인 +477k 매수 + 1년 +701% 모멘텀** — 턴어라운드 + 중국 테마 결합

### 🐻 Bear Case — "메모리 사이클 + 중국 리스크 + PBR 4.74x + 폭등"

◉ **메모리 사이클 변동성** — 다운사이클 전환 시 적자 재발. 2024년 OP -45억 경험. 턴어라운드 지속성 미검증
◉ **PBR 4.74x + 1년 +701% 폭등** — 적자 BPS 기준 고PBR. 단기 -20\~40% 조정 위험. 변동성 극단(BB 243.6%)
◎ **중국 자본 의존 + 지정학 리스크** — 동심반도체 30.20%, 미·중 반도체 갈등 영향. 수출 규제 리스크
◎ **소형 사이클·테마주** — 커버리지 부재, 변동성 매우 큼. 모멘텀 소멸 시 급락
○ **무배당** — 흑전에도 주주환원 부재

## 9. 종합 판단

**기업의 체질**: 피델릭스는 **메모리 반도체 Fabless + 중국 동심반도체(Dosilicon) 자회사**다. DRAM·NOR Flash·PSRAM·MCP 설계·판매. **2026Q1 OP 43억(+793.8%, OPM 20.0%) 폭발적 턴어라운드** — 7분기 적자 졸업. 중국 메모리 굴기 + 메모리 사이클 회복 + 중국 동심반도체 시너지가 핵심. 1년 +701% 폭등 + 외국인 +477k 매수.

**투자 가치가 있는 조건**:

| 조건 | 시점 | 확인 방법 |
|------|------|-----------|
| 2026Q2 OP 흑자 유지 (40억+) | 2026Q2 (8월) | 1Q26 턴어라운드 지속성 검증 |
| 중국 동심반도체 신규 수주 | 2026 | 모회사 연계 물량 확대 |
| 메모리 ASP 상승 지속 | 분기별 | NOR Flash·DRAM 가격 추이 |
| Automotive 메모리 매출 | 2026\~27 | 차량용 신규 수주 |

**핵심 모니터링**:
- ✅ **분기 OP 흑자 유지** — 2026Q1 43억 → 2\~3분기 지속성이 핵심
