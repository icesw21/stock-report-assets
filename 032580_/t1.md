## 2. 사업 구조

### 핵심 사업: 메모리 반도체 팹리스 + 중국 동심반도체 자회사

피델릭스는 1990년 설립, 1997년 코스닥 상장 — **메모리 반도체 (DRAM, Flash Memory, MCP) 설계 팹리스 specialist**. **중국 동심반도체(Dosilicon) 30.20% 모회사** — 중국 메모리 그룹 자회사.

**제품 포트폴리오**:
- **DRAM**: 모바일·IoT용 DRAM
- **Flash Memory (NOR·NAND)**: 모바일 버퍼 메모리
- **MCP (Multi-Chip Package)**: DRAM + Flash 통합 패키지
- **PSRAM (Pseudo SRAM)**: 모바일·IoT 향 신규
- **Mobile DDR + Serial NOR Flash**: 자동차 시장 진출 가속

**대상 시장**:
- **Consumer Electronics**: 모바일 폰 버퍼 메모리
- **M2M·IoT**: 사물인터넷 디바이스 향
- **Automotive**: 자동차 시장 신규 진출 — 차량용 ASIL-B 등급 인증

### 비즈니스 특성

- **B2B 팹리스 + 파운드리 위탁 생산**: 자체 설계 + 외부 생산
- **분기 매출 가속**: 97~183억 범위, 25Q4 183억
- **2025 매출 +19.6% V자 회복** (그러나 OP 적자 -21억 지속)
- **중국 동심반도체 30% 모회사** — 자금·기술 시너지

### 고객 구조 & 원가 구조

- **모바일 폰 OEM** (메인 — 삼성·LG·중국 OEM)
- **IoT·자동차** (확대)
- **글로벌 모바일 메모리 사이클 종속**

## 4. 성장 동력 분석

**동력 1: 모바일 메모리 회복 사이클 + 25Q4 흑전 ⭐⭐⭐ (◉)**
**2025 매출 +19.6% V자 + 25Q4 OP 흑전 +10억** — 모바일 메모리 사이클 회복 본격화. 글로벌 모바일 폰 + IoT 디바이스 메모리 수요 증가.

**동력 2: 중국 동심반도체 30% 모회사 시너지 ⭐⭐⭐ (◎)**
**중국 동심반도체(Dosilicon)** 메모리 그룹 자회사. 자금·기술 시너지 + 중국 시장 직접 침투 + 중국 메모리 산업 굴기 후방. SK·삼성 대비 약점인 중국 OEM 향 진입 가능.

**동력 3: 자동차 메모리 시장 진출 ⭐⭐⭐ (◎)**
**자동차 시장 신규 진출** (PSRAM·Mobile DDR·Serial NOR Flash). 자동차 메모리 시장 2030 ~$10B (CAGR 20%+). ADAS·자율주행 ECU 향 메모리 수요 폭증.

**동력 4: 제품 포트폴리오 다각화 (PSRAM·Mobile DDR·NOR Flash) ⭐⭐ (○)**
기존 DRAM·Flash·MCP에서 PSRAM·Mobile DDR·Serial NOR Flash로 확장 — 매출 다각화.

**동력 5: PBR 2.23x 적정 영역 + 단기 모멘텀 ⭐ (○)**
PBR 2.23x는 팹리스 평균 영역. 단기 +278% 폭등으로 시장 관심 회복.
