# 피델릭스 T4 Deep Dive — 2026Q1 OP 폭발 턴어라운드 + 중국 동심반도체 메모리 굴기 (메모리 Fabless 7분기 적자 졸업, NOR Flash·DRAM 사이클 회복)

---

## Executive Summary

피델릭스(032580)는 **메모리 반도체 Fabless + 중국 동심반도체(Dosilicon) 자회사**다. DRAM·NOR Flash·PSRAM·MCP·Mobile DDR 설계·판매. 1990년 설립, 1997년 코스닥 상장, 2017년 중국 동심반도체가 인수(최대주주 30.20%). 수출 비중 71% (일본·중국·유럽·미주).

핵심 투자 논리는 **3중 결합**: ① **2026Q1 OP 43억(+793.8%, OPM 20.0%) 폭발적 턴어라운드** — 7분기 연속 적자 졸업 ② **중국 동심반도체 시너지 + 메모리 굴기** — 중국 NOR Flash·DRAM 국산화 + 생산 capa + 글로벌 판매망 ③ **메모리 사이클 회복** — 다운사이클 바닥 → 업사이클 ASP 상승.

2024년 OP -45억 적자 바닥 → 2025Q4 +10억 흑전 → 2026Q1 +43억 폭발. 매출 215억(+97.5%, 사상 최대). PBR 4.74x는 적자 BPS 착시이며, **2026 OP 160억 추정 시 EPS 540원 → PER 12.6x**. 1년 +701% 폭등 + 외국인 +477k 매수.

리스크는 **메모리 사이클 변동성(적자 재발) + 중국 자본·지정학 리스크 + PBR 4.74x + 1년 +701% 폭등 + 변동성 극단(BB 243.6%)**. 다만 CB/BW 0건 + 유동비율 502.6%로 재무 양호.

**확률가중 적정주가 7,905원** (현재 6,810원 **+16.1%**) — 보수 4,800원(35%) / 기본 8,500원(45%) / 낙관 12,000원(20%). 전략: **추격 매수 비추, 단기 조정 시 MA20 4,057원 ~ 5,320원 분할 매수, SL 4,000원, 목표 8,500원**.

## 1. 분석 배경

피델릭스 주가는 2025년 저점 850원 → 2026년 8,460원으로 **1년 +701% 폭등** 후 6,810원. 2026Q1 OP 43억(+793.8%) 폭발적 실적 + 중국 메모리 굴기 테마 + 외국인 +477k 매수가 랠리 동력. 7분기 적자에서 흑자 폭발로 극적 전환.

T1~T3에서 확인한 사실:
- 2026Q1 OP 43억(+793.8%, OPM 20.0%) — 7분기 적자 졸업
- 매출 215억(+97.5%, 사상 최대)
- 중국 동심반도체 30.20% 최대주주 (메모리 굴기)
- 메모리 사이클 회복 (NOR Flash·DRAM·PSRAM)
- PBR 4.74x 적자 착시 (2026E PER 12.6x)

**왜 이 주제가 핵심인가?** — 메모리 Fabless는 ① 메모리 사이클(다운/업) ② 설계·제품 경쟁력 ③ 생산·판매 파트너의 3가지가 실적을 결정한다. 피델릭스는 메모리 업사이클 진입 + 중국 동심반도체 시너지(생산·판매·중국 내수) + NOR Flash/DRAM 제품군이 동시 결합하며 7분기 적자에서 흑자 폭발로 전환했다. **턴어라운드 지속성(2026Q2) + 중국 시너지 가시화 + 메모리 사이클 지속이 핵심**인 사이클·턴어라운드 베팅.

## 2. 산업/시장 개요

### 글로벌 메모리 반도체 사이클 (2024\~2027)

**메모리는 전형적 사이클 산업**. 2022\~24년 다운사이클(공급과잉 + 수요 둔화)로 가격 급락 → 2025\~27년 업사이클(AI·IoT·자동차 수요 + 감산 효과)로 ASP 회복. NOR Flash·DRAM·PSRAM 모두 업사이클 진입.

### 중국 메모리 굴기 + 동심반도체

**중국 반도체 자립 정책의 핵심 = 메모리 국산화**. 미·중 반도체 갈등 속 중국은 NOR Flash·DRAM·NAND 국산화 가속. **동심반도체(Dosilicon)는 중국 메모리 설계·생산 기업**으로, 2017년 피델릭스 인수 → 한국 설계 기술 + 중국 생산 capa + 중국 내수 시장 결합.

| 항목 | 동심반도체 시너지 |
|------|------|
| 생산 capa | 중국 파운드리(SMIC 등) 우선 할당 |
| 판매망 | 중국 내수 + 글로벌 대리점 |
| 기술 | 피델릭스 설계 IP + 동심 양산 |
| 시장 | 중국 메모리 국산화 수요 |
| 정책 | 중국 반도체 자립 보조금 |

### NOR Flash·DRAM·PSRAM 제품군

| 제품 | 용도 | 시장 동력 |
|------|------|----------|
| **NOR Flash** | 코드 저장 (IoT·자동차·가전) | IoT 디바이스 폭증 + 자동차 전장화 |
| **DRAM** | 휘발성 메모리 | AI·서버·모바일 수요 |
| **PSRAM** | 의사 SRAM (저전력) | IoT·웨어러블 |
| **Mobile DDR** | 모바일 메모리 | 스마트폰·태블릿 |
| **MCP** | 멀티칩 패키지 | 공간 효율 (모바일) |

### 시장의 특수성

| 구분 | 일반 반도체 | 메모리 Fabless |
|------|---------|-----------|
| 사이클 | 보통 | **극심 (다운/업 변동)** |
| 가격 변동성 | 보통 | **매우 큼 (ASP)** |
| 진입 장벽 | 높음 | **중 (설계 + 파운드리)** |
| 파트너 의존 | 보통 | **높음 (파운드리·판매)** |
| 지정학 영향 | 보통 | **큼 (미·중 갈등)** |

**핵심**: 메모리는 사이클 산업으로 다운/업 변동이 극심. 업사이클 진입 시 적자→흑자 폭발, 다운사이클 시 흑자→적자. 중국 동심반도체 시너지가 사이클 변동성 일부 완충.

## 9. 결론

피델릭스는 **메모리 반도체 Fabless + 중국 동심반도체(Dosilicon) 자회사**로, ① 2026Q1 OP 43억(+793.8%, OPM 20.0%) 폭발적 턴어라운드 ② 중국 동심반도체 시너지 + 메모리 굴기 ③ 메모리 사이클 회복(NOR Flash·DRAM·PSRAM)의 3중 결합이 사이클·턴어라운드 베팅 트리거.

**7분기 연속 적자 → 2025Q4 흑전 → 2026Q1 OP 43억 폭발**의 극적 전환. 매출 215억(+97.5%, 사상 최대). PBR 4.74x는 적자 BPS 착시이며, 2026 OP 160억 추정 시 EPS 540원 → PER 12.6x. 중국 메모리 굴기 + 동심반도체 생산·판매·자금 시너지가 사이클 변동성 완충.

단기적으로는 1년 +701% 폭등 + RSI 71.6 + 변동성 극단(BB 243.6%)으로 -20\~40% 조정 압력. 중장기적으로는 **메모리 업사이클 + 중국 시너지 + Automotive 신시장** 결합으로 **기본 8,500원(+25%) ~ 낙관 12,000원(+76%) 진입 여지**.

**핵심 트레이딩 시그널**: ① 추격 매수 강력 비추, 단기 조정 시 MA20 4,057원 ~ 5,320원 분할 매수 ② 2026Q2 실적(8월) OP 40억+ 유지 = 턴어라운드 검증 + 본격 비중 확대 ③ 중국 동심반도체 신규 수주 가시화 ④ 메모리 ASP 사이클 지속.
