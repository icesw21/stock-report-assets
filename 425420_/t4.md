# 티에프이 T4 Deep Dive — 칩이 바뀌면 테스트도 바뀐다: AI·CPO 테스트 솔루션 국내 유일 토탈벤더

## Executive Summary

티에프이는 **Test Board·Test Socket·COK를 4개 테스트 공정(Final Test·신뢰성·SLT·SET 모사)에 모두 공급하는 국내 유일** 반도체 테스트 솔루션 기업이다.

- **현재가 50,300원** (2026-06-09, 당일 **+18.35%**), 시가총액 약 **6,211억원**
- **폭발 성장**: 2025 매출 +51.8%·영업이익 +334%·OPM 17.1%·ROE 16.6%
- **성장 동력**: AI 반도체·HBM·**CPO(Co-Packaged Optics)** 고난도 테스트 + **SLT(시스템 레벨 테스트)** 확대
- **밸류**: PER 31.96x(TTM)·Forward PER \~26x. 고점(74,400원) 대비 -32% 조정 후 당일 +18% 급반등
- **확률가중 적정주가 57,150원** (현재가 **+13.6%**)

종합 의견: "칩이 바뀌면 테스트도 바뀐다" — 칩 복잡도 상승으로 테스트 강화 수혜. 국내 유일 4공정 토탈솔루션 + 폭발 성장 + 안정 지배구조(최대주주 69.5%)의 성장주. 당일 +18% 급등 직후로 **눌림목 분할 매수** 권고.

## 2. 산업/시장 개요

### 반도체 테스트 공정

반도체는 조립 후 ① Final Test(최종 검사), ② 신뢰성 테스트, ③ SLT(System Level Test, 시스템 레벨), ④ SET 모사 Test 등 다단계 테스트를 거친다. 각 공정에 Test Board·Socket·COK가 필요하며, 칩이 고도화될수록 테스트 단계·난이도가 증가한다.

### "칩이 바뀌면 테스트도 바뀐다"

| 칩 트렌드 | 테스트 영향 |
|---------|----------|
| AI 반도체·HBM | 고난도·고속 테스트, 소켓·보드 스펙↑ |
| CPO(Co-Packaged Optics) | 광-전 통합 패키지 테스트, 수율 확보 핵심 |
| 칩 복잡도 상승 | SLT 확대, 테스트 단계 증가 |

AI·HBM·CPO 등 첨단 칩은 테스트 난이도·단계가 늘어, 테스트 소켓·보드 ASP·물량이 동반 증가한다. "CPO 수율 잡기, 테스트가 돈이 된다"는 평가다.

### CPO·SLT — 차세대 테스트 수요

CPO(Co-Packaged Optics)는 AI 데이터센터의 광-전 통합 패키징으로, 수율 확보를 위한 테스트가 핵심이다. SLT(시스템 레벨 테스트)는 칩을 실제 시스템 환경에서 검증하는 고도화 테스트로, 칩 복잡도 상승과 함께 확대된다. 티에프이는 이 차세대 테스트 수요의 직접 수혜주다.

## 9. 결론

티에프이는 Test Board·Socket·COK를 Final Test·신뢰성·SLT·SET 모사 4개 공정에 모두 공급하는 **국내 유일 반도체 테스트 토탈솔루션** 기업이다. 2024 둔화 후 2025 매출 +51.8%·영업이익 +334%·OPM 17.1%·ROE 16.6%로 폭발 성장했다. **AI 반도체·HBM·CPO(Co-Packaged Optics) 고난도 테스트 + SLT(시스템 레벨 테스트) 확대**가 성장 동력으로, "칩이 바뀌면 테스트도 바뀐다"·"CPO 수율 잡기, 테스트가 돈이 된다"는 평가다. 소켓·COK 소모성 반복 매출 + 최대주주 69.47% 안정 지배 + CB 0건의 견조한 구조.

당일 +18.35% 급등은 과매도(스토캐 18.5) 반등이며, 고점(74,400원) 대비 -32% 조정 후 반등 국면이다. Forward PER \~26x는 폭발 성장·ROE 16.6%·국내 유일 지위 대비 합리적이나, 테스트 수요 사이클·2026Q1 OPM 둔화·급등 변동성이 리스크다. 확률가중 적정주가 57,450원은 +14% 업사이드다.

**당일 급등 추격보다 MA120·49,000원 ~ MA200 45,768원 눌림목 분할 매수**가 합리적이며, AI·CPO 테스트 수주·SLT 확대·OPM 회복을 확인하며 비중을 늘리는 전략을 권고한다.
