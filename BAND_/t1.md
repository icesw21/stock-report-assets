## 2. 사업 구조

### 사업 개요

**밴드위스는 미국 CPaaS(Communications Platform-as-a-Service) 클라우드 통신 기업**이다. 기업이 음성·메시징 통신을 모바일 앱·연결 기기에서 구축·확장·운영하도록 지원. **SIP·프로그래머블 보이스 API + 메시징 API + E911 긴급서비스 + 번호 관리 + 캐리어 통합(UCaaS/CCaaS/Conversational AI/STT-TTS)**. 캐리어급 자체 네트워크 인프라 직접 보유 차별화.

### 사업부문별 매출 구조 (FY2025 추정)

| 사업부문 | 주요 서비스 | 비중(%) | 비고 |

| Cloud Communications | 음성·메시징 API, E911 | \~80 | 핵심 CPaaS |
| 번호 관리·캐리어 통합 | UCaaS/CCaaS 통합 | \~15 | |
| AI 음성·신규 | Conversational AI, STT/TTS | \~5 | 성장 |
| **합계** | | 100.0 | 매출 \~$750M |

※ FY2025 추정. CPaaS 단일 플랫폼. 단위: 서비스별 추정 비중.

### 핵심 사업: CPaaS 클라우드 통신 (캐리어 인프라 + AI 음성)

**기업용 클라우드 음성·메시징 API 플랫폼**. Twilio·Sinch와 경쟁하나 **캐리어급 자체 네트워크 인프라 직접 보유** + E911 긴급서비스 차별화. AI 음성(Conversational AI·STT/TTS) 수요 수혜.

- **Cloud Communications**: SIP·프로그래머블 보이스 API + 메시징 API + E911. 엔터프라이즈 통신 인프라
- **번호 관리·캐리어 통합**: UCaaS/CCaaS 통합, 번호 포팅·관리
- **AI 음성 (신규)**: Conversational AI + STT/TTS. AI 콜센터·음성 봇 인프라

### 비즈니스 특성

- **흑전 전환 진행**: 적자(OP margin -2.2%, EPS -$0.38) → **forward EPS +$2.24 흑전 전망**. 매출 성장 + 마진 개선. Forward PE 28.3x
- **매출 성장 (CPaaS)**: 2021 $491M → 2024 $748M(+24.5%) 성장. 2025 보합($750M). AI 음성·엔터프라이즈 수요
- **캐리어 인프라 차별화**: Twilio(인프라 임대) 대비 자체 캐리어 네트워크 보유 → 마진·안정성 우위. E911 규제 해자
- **1년 +407% 급등 + 과매수**: $12.82→$64.97. CPaaS·AI 음성 + 흑전 기대. RSI 79 과매수

### 고객 구조 & 원가 구조

- **고객 구조**: 엔터프라이즈·SaaS·콜센터 (B2B). Google·Microsoft·Zoom 등 대형 통신 SaaS 백엔드. AI 음성 기업
- **매입/원가**: 네트워크 인프라 운영 + 캐리어 비용. 자체 인프라로 마진 우위. R&D + 부채 이자(부채비율 163%)

## 4. 성장 동력 분석

**동력 1: AI 음성 인프라 (Conversational AI·STT/TTS) ⭐⭐⭐⭐ (◉)**
**AI 콜센터·음성 봇·Conversational AI 인프라 수요 폭증**. STT(음성→텍스트)/TTS(텍스트→음성) + AI 음성 플랫폼. 밴드위스 캐리어 인프라가 AI 음성 백엔드. AI 에이전트·음성 AI 시대 직접 수혜. 핵심 성장 동력.

**동력 2: 흑전 전환 + 마진 개선 ⭐⭐⭐⭐ (◉)**
적자(OP margin -2.2%) → **FY2026 흑전(forward EPS +$2.24)**. 매출 성장 + 캐리어 인프라 마진 + 비용 효율. 흑전 정상화 + 이익 성장. Forward PE 28.3x.

**동력 3: 캐리어 인프라 차별화 + E911 해자 ⭐⭐⭐ (◎)**
Twilio(인프라 임대) 대비 **자체 캐리어 네트워크 직접 보유** → 마진·안정성 우위. **E911 긴급서비스 규제 해자** (라이선스·인증). 엔터프라이즈 통신 락인.

**동력 4: CPaaS 시장 성장 + 엔터프라이즈 ⭐⭐⭐ (◎)**
글로벌 CPaaS 시장 성장 + 엔터프라이즈 클라우드 통신 전환. Google·MS·Zoom 등 대형 SaaS 백엔드. UCaaS/CCaaS 통합 수요.
