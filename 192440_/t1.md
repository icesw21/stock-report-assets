### 핵심 사업: 글로벌 모바일 액세서리 브랜드 'Spigen' (아마존 1위)

**2009년 설립, Spigen 브랜드로 스마트폰 케이스·보호필름을 아마존 등 글로벌 이커머스에서 판매하는 모바일 액세서리 기업**. 북미에서 시작해 유럽·인도·일본으로 확장, **아마존 모바일 케이스 카테고리 글로벌 1위권**. 자체 특허 기술 + 온라인 마케팅 강점.

- **국내**: 회사물류센터 → 오픈마켓·종합몰·자사몰·대리점. 국내 매출 7.9% (성장 중)
- **해외**: 한국 → 종속법인(미국·유럽·인도·일본) → 아마존·오프라인 딜러. 수출 92.1%, 북미 41.5% + 유럽 32.3%
- **신사업**: ① 차량용 액세서리(거치대·충전기) ② 프리미엄 충전기(GaN) ③ 폴더블 스마트폰 케이스 ④ 웨어러블(워치·버즈) 액세서리 — 카테고리 확장

### 비즈니스 특성

- **아마존 D2C 글로벌 브랜드**: 아마존 베스트셀러 중심 시장 리더십. 자사 브랜드(Spigen) D2C로 마진·브랜드 가치 확보. 수요 발생 즉시 발주(수주 개념 없음)
- **저PBR 가치주 + 순현금**: PBR 0.37x (BPS 91,806원) + 유동비율 862.4% + 순현금 구조. 시총 2,123억 중 현금성 비중 높음
- **마진 회복 진행**: 2024년 OP 217억(저점) → 2025년 332억(+52.7%) → 2026Q1 112억(+21.5%). 원가·물류비 안정 + 환율
- **폴더블·신제품 사이클**: 갤럭시 Z폴드/플립 + 아이폰 신제품 출시 사이클에 케이스·필름 수요 동조

### 고객 구조 & 원가 구조

- **고객 구조**: B2C(D2C). 아마존 글로벌 + 자사몰 + 오프라인 딜러. **단일 고객 10% 이상 없음** (분산). 북미·유럽 중심
- **매입/원가**: 외주 생산(중국·베트남 OEM) + 자체 디자인·특허. 환율(수출 92%) + 물류비 + 아마존 수수료가 마진 변수

## 4. 성장 동력 분석

**동력 1: 마진 회복 (OPM 4.7% → 8%+) ⭐⭐⭐⭐ (◉)**
2022년 OP 520억(OPM 11.7%) → 2024년 217억(OPM 4.7%) 바닥 → **2025년 332억(+52.7%) → 2025Q4 OPM 10.7% + 2026Q1 OPM 9.9% 회복**. 원가·물류비 안정 + 환율 우호 + 신제품(폴더블·GaN 충전기) 고마진 믹스. 2026 OP 400억(OPM 8.3%) 추정 — 이익 정상화가 핵심.

**동력 2: PBR 0.37x + 순현금 3,000억 가치주 ⭐⭐⭐⭐ (◉)**
BPS 91,806원 대비 PBR 0.37x. **현금성 3,200억 > 시총 2,123억** — 순현금이 시총보다 큰 극단 저평가. 자사주 7.0% + 배당 2.19%. 정부 밸류업 + 주주환원(자사주·배당 확대) 시 강력한 리레이팅. PBR 0.5x만 회복해도 +35%.

**동력 3: 신제품 카테고리 확장 (폴더블·GaN·차량용) ⭐⭐⭐ (◎)**
스마트폰 케이스(53.8%) 중심에서 **폴더블 케이스 + GaN 프리미엄 충전기 + 차량용 거치대·충전기 + 웨어러블 액세서리**로 확장. 갤럭시 Z폴드/플립 + 아이폰 폴더블(2026\~27 예상) 출시 시 폴더블 케이스 수요. 충전기·차량용 고마진 카테고리.

**동력 4: 유럽·국내 성장 + 인도 신시장 ⭐⭐ (◎)**
유럽(+9.0%) + 국내(+16.5%) 성장. 인도 신시장 진출 + 일본 확장. 북미(41.5%) 의존 분산 진행. 글로벌 이커머스(아마존) 판매망 + 자사몰 D2C 강화.

## 8. Bull vs Bear

### 🐂 Bull Case — "PBR 0.37x + 순현금 3,000억 + 마진 회복"

◉ **PBR 0.37x + 순현금 3,000억 > 시총 2,123억** — 순현금이 시총보다 큰 극단 저평가. EV 기준 본업 거의 무료. PBR 0.5x만 회복해도 +35%
◉ **마진 회복 (OPM 4.7% → 10%+)** — 2024 OP 217억 바닥 → 2025 332억 → 2026Q1 OPM 9.9%. 이익 정상화
◎ **자사주 7.0% + 배당 2.19% 주주환원 잠재** — 밸류업 + 자사주 소각·배당 확대 시 강력한 리레이팅
◎ **신제품 카테고리 (폴더블·GaN·차량용)** — 케이스 중심에서 고마진 카테고리 확장
○ **글로벌 브랜드 Spigen + 아마존 1위** — 브랜드 가치 + 유럽·국내·인도 성장

### 🐻 Bear Case — "매출 정체 + 시장 성숙 + 북미 관세 + 환율"

◉ **매출 정체 (4,600억대) + 시장 성숙** — 스마트폰 판매 둔화 + 케이스 시장 성숙. 성장성 제한
◉ **북미 매출 41.5% + 트럼프 관세** — 중국·베트남 OEM 생산 → 미국 수출 관세 부담 가능
◎ **환율 변동 (수출 92%)** — 원화 강세 시 수출 마진 압박
◎ **아마존 플랫폼 의존** — 수수료·정책 변화 리스크
○ **52주 신고가 + RSI 75.7 과매수** — 1년 +59% 후 단기 -10\~15% 조정
