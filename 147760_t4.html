피엠티 — 2D MEMS NAND 프로브카드 수율 안정화와 점유율 회복 분석

Executive Summary

피엠티는 2021년 품질 이슈 이후 3D→2D MEMS 공정 전환 과정에서 4년간 적자를 기록했으나, 2025년 8월 2D 납품을 재개하며 턴어라운드 국면에 진입했다. 경쟁사(코리아인스트루먼트, 티에스이)가 HBM/DRAM 테스트 영역에 리소스를 집중하면서 NAND 프로브카드 시장에 구조적 공백이 발생했고, 동사는 과거 50% 점유율 기반의 레퍼런스를 보유하고 있다. 현재 수주잔고 270억원은 분기 BEP(130~140억)를 상회하는 수준으로, 2026년 상반기 2D 핀 수율 85% 이상 달성 시 납품 물량 확대와 원가 개선이 동시에 실현되며 흑자 전환이 가능하다. 확률가중 적정주가 4,120원 대비 현재가(6,160원)는 이미 성공 시나리오를 상당 부분 선반영한 상태다.

1. 분석 배경

T1에서 식별한 최우선 성장동력(⭐⭐⭐)은 2D MEMS NAND 프로브카드의 수율 안정화와 점유율 회복이다. 이 주제가 핵심인 이유는 다음과 같다.

  • TAM: 국내 NAND 프로브카드 시장 약 1,000억원. 동사 최대 50% 점유 이력 → 회복 시 연매출 500억+ 가능
  • 실적 기여 시점: 2025년 8월 납품 재개, 현재 수주잔고 270억(2026년 상반기 기준). 수율 안정화 달성 시 2026년 하반기부터 분기 BEP 달성 현실적
  • 리레이팅 임팩트: 흑자 전환 + 점유율 회복 확인 시 PBR 2.0x 이상으로 리레이팅 가능 (현재 2.15x는 기대 프리미엄 과도 선반영)
  • 차별화: 국내에서 NAND 전용 2D MEMS 프로브카드를 공급할 수 있는 유일한 기업으로, 수율 안정화 성공 시 경쟁 공백에서 독점적 지위 회복 가능

분석 범위

  • NAND 프로브카드 시장 구조 및 수급 동향
  • 2D MEMS 공정 전환 현황 및 수율 개선 로드맵
  • 경쟁사 동향과 NAND 시장 공백의 지속성
  • 수율 안정화 시나리오별 실적 기여 및 밸류에이션 영향

한계

  • 2D 핀 수율 수치는 비공개 — 미래에셋증권 탐방노트(2026.02) 기반 추정
  • NAND 시장 점유율 데이터는 독립 통계 부재로 과거 레퍼런스 기반
  • 고객사(특정 메모리 업체) 발주 계획 미공개

2. 산업/시장 개요

NAND 프로브카드 시장 현황

NAND 플래시 메모리의 공정 미세화(176단 → 200단+)와 셀 집적도 증가에 따라 웨이퍼 테스트의 중요성이 높아지고 있다. 프로브카드는 웨이퍼 테스트 공정에서 개별 다이의 전기적 양/불량을 판정하는 핵심 소모품으로, 공정 세대 전환 시마다 교체 수요가 발생한다. NAND 영역에서는 DRAM 수준의 핀 밀도가 요구되면서 고정밀 어드밴스드 MEMS 방식으로의 전환이 업계 트렌드가 되었다.

국내 NAND 프로브카드 시장은 특정 메모리 업체를 중심으로 4,000~4,500억원 규모로 추정되며, NAND 영역은 약 25%(1,000억원)에 해당한다. 피엠티는 이 시장에서 과거 약 50%의 점유율을 보유했으나 2D 전환 품질 이슈로 공급이 중단된 상태였다.

4. 피엠티의 역량 분석

진입 전략

피엠티는 2D MEMS 방식으로의 기술 전환을 완료하고 2025년 8월 납품을 재개했다. 고객사 내 NAND 테스트 캐파 중 2D 프로브카드 비중을 단계적으로 확대하는 방식으로 재진입 중이며, 현재 2D 핀 수율 안정화(목표: 경쟁사 기준 85~87%)를 위해 개선 작업을 진행 중이다.

  • 2D MEMS 전용 공정: NAND에도 DRAM 수준의 핀 밀도가 요구되는 트렌드에서, 2D MEMS는 기술적으로 불가피한 방향. 피엠티는 이미 공정 전환 완료
  • 경쟁 공백 타이밍: 티에스이·코리아인스트루먼트가 HBM에 집중하는 시기에 NAND 전용 공급자로 재진입

6. 밸류에이션 영향

PBR 리레이팅 논리

현재 PBR 2.15x는 수율 안정화 기대감이 반영된 수준이나, 실제 BPS(2,859원)는 적자 지속으로 계속 하락 중이다. 수율 안정화 및 흑자 전환 확인 시 BPS 하락이 멈추고 PBR 리레이팅이 가능하다. Peer(JEM 2.1x, 티에스이 2.6x) 수준을 기준으로 시나리오를 설정했다.

9. 결론

피엠티의 2D MEMS NAND 프로브카드 수율 안정화는 기업 생사를 결정하는 단일 핵심 변수다. 2025년 8월 납품 재개와 수주잔고 270억원은 긍정적 신호이나, 2023년에도 초도 납품 후 품질 실패로 중단된 전례가 있어 현재 수율 진행 상황의 면밀한 검증이 필요하다. 경쟁사의 NAND 공백은 실적 회복을 위한 유리한 환경을 조성하고 있으며, 수율 안정화 달성 시 분기 BEP(130~140억) 이상의 매출이 가능한 수주잔고가 이미 확보된 상태다. 그러나 현재 주가(6,160원)는 확률가중 적정주가(4,772원) 대비 약 29% 고평가 상태로, 수율 성공을 이미 반영하고 있어 리스크/리워드는 불리하다.

모니터링 체크포인트:
- ✅ 2026년 Q1/Q2 분기 매출 130억원 돌파 여부 — 수율 안정화의 실적 확인
- ✅ 2D 핀 수율 85%+ 달성 공식 언급 (IR 또는 공시) — 리레이팅 트리거
- ✅ 현금잔고 및 유동성 공시 — 자금조달 이슈 발생 시 주가 급변 가능
