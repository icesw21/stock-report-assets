# 티에이치엔(019180) — 현대차 SUV/HEV 신차 고단가 사이클 + 제이에스엔 영업양수 시너지: PER 2.5x 극저평가의 리레이팅 베팅

---

## Executive Summary

티에이치엔은 현대차 와이어 하네스 3위 Tier-1(점유율 13%)로, 2025년 매출 9,891억(+52%)·OP 701억(+126%)·ROE 30.4% 폭발 성장을 시현했음에도 PER 2.54x·PBR 0.78x로 한국 자동차부품 평균(7\~17x) 대비 1/3\~1/6 극단 할인 거래 중이다. 폭발의 핵심은 **(1) 현대차 SUV/HEV 신차 사이클 고단가화** — 팰리세이드HEV 1,777천원/대 vs 일반 산타페 604천원/대(2\~3배 단가), **(2) 2025.5 제이에스엔 와이어 하네스 영업양수**(현금 98억) — 분기 +300억 매출 + OPM 2→8% 점프 두 축이다. 2026Q1 OP 175억(+417%) 지속, 2026\~2028 셀토스 신형·HB20 신형·아이오닉/EV9 ramp·EV/자율주행 침투 가속 시 사이클 연장 가능성 높다. 보수 30% / 기본 50% / 낙관 20% 확률가중 적정주가는 **32,200원 (현재 9,680원 대비 +233%)**. PER 5x · PBR 1.0x 회복만으로도 +60\~130% 업사이드. 단일 고객 의존 + 1년 +256% 단기 과열은 분할 매수로 대응.

## 1. 비즈니스 모델

### 1.1 사업 구조 및 매출 흐름

티에이치엔은 1986년 설립 40년차 자동차용 와이어링 하네스(Wiring Harness) 전문 제조사. 자동차 한 대당 수천 가닥의 전선·커넥터를 차종별로 설계·조립해 OEM에 직납하는 Tier-1 부품사다. 2025년 연결 매출 9,891억원 중 와이어 하네스 제품이 9,826억(99.3%)·설계 용역 66억(0.7%)으로 단일 제품 집중도가 극단적으로 높다. 매출의 59.4%(5,875억)는 수출(파라과이·브라질·중국·필리핀·베트남 5개국 현지 법인 경유 + 본사 직수출), 40.6%(4,016억)는 내수(현대차/기아/현대모비스 직납)다.

### 1.2 고객·제품 구조

- **국내**: 현대차/기아 Tier-1. 주력 차종 투싼·팰리세이드·산타페·아이오닉6·EV9·G70·셀토스 등 SUV·세단·EV 전반 커버
- **해외**: 파라과이(남미 허브, HB20·크레타 직납) · 브라질(현지 OEM 라인) · 중국·필리핀·베트남(부분 공정)
- **점유율**: 현대차 와이어 하네스 vendor 3위 **13%** (1위 경신 46% / 2위 유라코퍼레이션 41%)

### 1.3 비즈니스 모델의 핵심 변수

| 변수 | 메커니즘 | 2025년 실증 |
|------|---------|----------|
| 차종별 단가 차이 | SUV > 세단, HEV > ICE, EV > HEV (전장화 비례) | 팰리세이드HEV 1,777천원/대 vs 산타페 604천원/대 = **3배** |
| 모델 사이클 | 신차 출시·풀체인지 = 단가·물량 동시 점프 | 팰리세이드/팰리세이드HEV 신형(25 1H) · 산타페 연식변경(25 3Q) |
| 환율 (KRW/USD) | 수출 59% 구조, 원화 약세 = OPM 직접 수혜 | 2025년 환율 우호 + 매출 +52% 결합 |
| 구리 가격 (LME) | 원재료비 100% 당사 부담, 단가 패스스루 6\~12개월 지연 | 2024Q4 OP 29억 충당금 일회성 ← 구리 반영 지연 |
| 고객 OEM 노조·생산 | 현대차 한국 공장 가동률 = 직격 | 2025년 현대차 글로벌 +5% · 한국 정상가동 |

### 1.4 비즈니스의 약점과 강점

**약점**:
1. 현대차 단일 고객 의존(매출 90%+) — 글로벌 OEM(GM·Stellantis·Ford) vendor list 부재
2. 점유율 13% 3위 고착 — 경신 46% · 유라 41%와 협상력 격차
3. 단일 제품(와이어 하네스) 집중 — 모듈·전장 통합 트렌드에 대응 부재
4. 부가가치 낮은 노동집약 공정 — 인건비·환율 민감

**강점**:
1. 40년 현대차 동행 — 신차 개발 단계 동참, 사양 변경 대응력
2. 파라과이·브라질 남미 허브 — 멕시코 IRA 우회 + 현지 OEM 진입로
3. 가족 지배(특수관계인 42.8%) — 단기 의사결정 빠름, 양수합병 결단력 입증(제이에스엔)
4. 부채비율 171% · 순차입금비율 7.3% — 사실상 무차입 + FCF 537억 보유

## 4. 경쟁력 및 해자(Moat)

### 4.1 해자의 본질

티에이치엔의 해자는 **"40년 현대차 신차 개발 동행"** 단 하나다. 이는 단순 거래관계가 아니라:
- 신차 컨셉 단계 동참 → 전기 사양·하네스 설계 합동
- 양산 단계 동시 ramp-up → JIT 납품 체계
- 사양 변경 시 6\~8주 내 대응
- 품질사고 시 무한 책임 — vendor 신뢰 누적

이 진입장벽은 신규 vendor 진입을 봉쇄한다. 경신·유라·티에이치엔 3사가 100% 점유율 유지하는 이유.

### 4.2 해자의 한계

| 한계 | 영향 |
|------|------|
| 단일 고객 의존 | 현대차 글로벌 판매 -10% = 매출 -10% 직격 |
| 글로벌 OEM vendor list 부재 | GM/Stellantis/Ford 진입 불가 → TAM 제한 |
| 모듈 통합 트렌드 미대응 | 현대모비스·만도가 모듈 단위 통합 추진 시 디스인터미데이션 가능 |
| 점유율 13% 3위 | 신차 우선배정·단가협상 불리 |

### 4.3 제이에스엔 영업양수 시너지 — 해자 보강

2025.5 관계사 ㈜제이에스엔의 와이어 하네스 사업부를 현금 98억으로 영업양수. 사업부 2023년 매출 1,400억 → 동사 인수 후 **분기 +300억 매출 효과**, 연간 약 1,200억 매출 추가 기대.

**시너지 메커니즘**:
1. 공통비 분산 — 본사 관리비·R&D 공유 → 사업부 OPM +3\~4%p
2. 원자재 협상력 — 구리·전선 발주 규모 확대 → 원가 -1\~2%p
3. 생산설비 통합 — 본사 라인·인력 공유 → 가동률 +5\~10%p
4. **결과**: 동사 OPM 2% → 8% 점프(2025Q1 → Q2~Q4)에 결정적 기여

## 5. Bull Case — 「PER 5x 회귀 + HEV 사이클 풀ramp」

### 5.1 핵심 가설

1. **현대차 SUV/HEV 사이클 2026\~2028 지속** — 셀토스 신형·HB20 신형·아이오닉/EV9 ramp로 매출 +7\~15%/년 유지
