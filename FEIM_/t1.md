## 2. 사업 구조

### 사업 개요

**프리퀀시 일렉트로닉스는 정밀 타이밍·주파수 제어(precision time & frequency) 제품 전문 기업**(뉴욕)이다. 통신위성·지상 통신국·미군·GPS용 초정밀 발진기·원자시계·타이밍 시스템. **2개 세그먼트: FEI-NY(통신위성·미군 정밀 타이밍) + FEI-Zyfer(GPS/PNT·보안통신)**. 우주·국방 PNT(Position·Navigation·Timing) 핵심 부품.

### 사업부문별 매출 구조 (FY2025 추정)

| 사업부문 | 주요 제품 | 비중(%) | 비고 |
|---------|---------|------:|-----|
| FEI-NY | 통신위성·미군 정밀 타이밍·발진기 | \~70 | 핵심, 위성·국방 |
| FEI-Zyfer | GPS/PNT·보안통신·로케이터 | \~30 | GPS 백업·PNT |
| **합계** | | 100.0 | 매출 \~$70M |

※ FY2025 추정. 위성·국방 타이밍 + GPS/PNT. 단위: 세그먼트 추정 비중.

### 핵심 사업: 우주·국방 정밀 타이밍 (위성·미군·PNT)

**초정밀 발진기·원자시계 — 통신위성·미군·GPS 백업(PNT)의 핵심 타이밍 부품**. 위성 통신·항법은 나노초급 타이밍 동기화 필수. 미군 통신·항법 + 저궤도 위성통신(LEO) 확대 + GPS 교란 대비 PNT 백업 수혜.

- **FEI-NY**: 통신위성 탑재 정밀 발진기·원자시계 + 미군 통신·레이더 타이밍. 위성·국방 수주
- **FEI-Zyfer**: GPS/PNT 시스템 — 보안통신·정부·상업. GPS 교란 대비 백업 타이밍(eLORAN 등)

### 비즈니스 특성

- **흑전 폭증 턴어라운드**: FY2023 적자(OP -$4.7M) → FY2024 흑전(OP $5.0M) → **FY2025 OP $11.7M·NI $23.7M(EPS $2.46) 폭증**. 단 FY2025 NI는 일회성(투자수익·세금) 일부 포함
- **위성·국방 PNT 수혜**: 저궤도 위성통신(LEO) 확대 + 미군 통신·항법 현대화 + GPS 교란 대비 PNT 백업. 우주·국방 타이밍 수요
- **소형 + 수주 변동성**: 매출 \~$70M 소형. 위성·국방 프로젝트 수주 시점에 따라 분기 변동. 4월 결산
- **1년 +315% 급등 + 과매수**: $18.26→$75.88. 우주·국방 테마 + 흑전. PER 101x + RSI 73.9 과매수

### 고객 구조 & 원가 구조

- **고객 구조**: 미 국방부·방산(록히드·노스롭 등 위성 OEM) + 정부 + 상업 위성·통신. B2B 수주
- **매입/원가**: 정밀 부품·인건비(엔지니어). 위성·국방 프로젝트 마진. 흑전 후 OP margin 16.8%(FY2025)

## 4. 성장 동력 분석

**동력 1: 우주·위성 타이밍 (LEO 위성통신 확대) ⭐⭐⭐⭐ (◉)**
**저궤도(LEO) 위성통신 확대** — 스타링크·아마존 쿠이퍼·국방 위성 등 위성 폭증. 위성 탑재 정밀 발진기·원자시계 수요. FEI-NY 위성 타이밍 수혜. 우주 인프라 투자 사이클 직접 수혜.

**동력 2: 국방 PNT + GPS 백업 ⭐⭐⭐⭐ (◉)**
**미군 통신·항법 현대화 + GPS 교란 대비 PNT(Position·Navigation·Timing) 백업**. 지정학 긴장 + GPS 재밍·스푸핑 위협으로 대체 타이밍(원자시계·eLORAN) 수요. FEI-Zyfer GPS/PNT + FEI-NY 군 타이밍. 국방 예산 증가 수혜.

**동력 3: 흑전 턴어라운드 + 수주 ⭐⭐⭐ (◎)**
FY2023 적자 → FY2025 흑전(OP $12M). 위성·국방 수주 회복 + 마진 개선. 단 FY2026 분기 둔화(OP margin 2.6\~10%)로 수주 지속성 검증 필요.

**동력 4: 정밀 타이밍 니치 + 진입 장벽 ⭐⭐⭐ (◎)**
초정밀 발진기·원자시계는 우주·국방 인증(고신뢰성) 진입 장벽 높음. FEIM 50년+ 트랙. 위성·국방 락인. 틈새 독과점.

## 9. 종합 판단

**기업의 체질**: 프리퀀시 일렉트로닉스는 **위성·국방 정밀 타이밍(발진기·원자시계·PNT) 소형 니치 기업**이다. FEI-NY(위성·미군) + FEI-Zyfer(GPS/PNT). **우주·LEO 위성통신 확대 + 국방 PNT·GPS 백업 수혜 + FY2023 적자 → FY2025 흑전(OP $12M)**. 단 FY2025 NI는 일회성 포함, 2026Q1~Q3 분기 둔화. PER 101x + 1년 +315% + RSI 73.9 과매수.

**투자 가치가 있는 조건**:

| 조건 | 시점 | 확인 방법 |
|------|------|-----------|
| 위성·국방 수주 회복 | FY2026~ | 분기 매출 + 수주잔고 |
| OP margin 회복 (15%+) | FY2026 | 분기 마진 정상화 |
| 정상 EPS 성장 (일회성 제외) | FY2026 | EPS $1.14+ 검증 |
| 우주·국방 PNT 매출 가속 | 2026~ | LEO·PNT 수주 |

**핵심 모니터링**:
- ✅ **위성·국방 수주잔고** — LEO 위성·미군 타이밍 수주 (성장 핵심)
- ✅ **OP margin 회복** — FY2026 2.6\~10% → 15%+ 정상화
- ✅ **정상 EPS (일회성 제외)** — FY2025 일회성 제외 정상 수익성
- ✅ **국방 PNT·GPS 백업** — 미군 항법 현대화 수주
- ✅ **PER 101x + 1년 +315%** — 일회성 착시 + 과매수 변동

⚠️ **프리퀀시 일렉트로닉스는 우주·국방 정밀 타이밍 + 흑전 펀더멘털은 유효하나, PER 101x(FY2025 일회성 착시) + 1년 +315% + FY2026 분기 둔화로 단기 고평가·과열입니다. 위성·국방 수주 회복·정상 수익성이 밸류 정당화 관건입니다.**
