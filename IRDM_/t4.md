# 이리듐 T4 Deep Dive — LEO 위성통신 선두 + D2D·IoT·국방 성장 (흑자·FCF·주주환원 vs PER 49.7x·부채 447% 부담)

---

## Executive Summary

이리듐(IRDM)은 **66개 LEO(저궤도) 위성 컨스텔레이션 글로벌 커버리지 위성통신 선두**다. 음성·데이터·IoT + D2D(직접위성통신) + 정부(국방). 스타링크(광대역)와 차별화된 음성·IoT·국방 틈새 + 극지방 포함 전 지구 커버리지.

핵심 투자 논리는 **D2D·IoT·국방 성장 + 흑자·FCF·주주환원**: ① **D2D(직접위성통신, 스마트폰)** — 퀄컴 협력, 스마트폰 위성 연결 ② **위성 IoT** — 저비용 자산 추적·M2M 고성장 ③ **국방(DoD) 안정 계약** ④ **흑자(ROE 24.7%) + 강한 FCF(\~$350M) + 주주환원**(배당 증가 + 자사주).

매출 성장(2021 $615M → 2025 $870M) + FY2024\~25 OP margin 24\~28%·ROE 24.7% 급등(위성 인프라 구축 완료 후 고마진). 1년 +215% 급등은 위성통신·D2D 테마. 단 PER 49.7x + 부채비율 447%(위성 CapEx) + RSI 77.5 과매수.

**확률가중 적정주가 $49.1** (현재 $51.78 **-5.2%, 소폭 고평가**) — 보수 $36(35%) / 기본 $50(40%) / 낙관 $66(25%). 전략: **MA20 $44 ~ $38 눌림목 분할 매수, D2D·IoT 성장 가시화 시 리레이팅**.

## 2. 산업/시장 개요

### 위성통신 + D2D 시장

**위성통신은 D2D(직접위성통신)로 재성장**. 스마트폰 직접 위성 연결(긴급·SMS·데이터) + 위성 IoT + 광대역(스타링크). 지상망 부재 지역 + 긴급 통신 + IoT 수요.

| 구분 | 2024 | 2027E | 2030E |
|------|----:|-----:|-----:|
| 글로벌 위성통신 ($B) | 30 | 50 | 90 |
| D2D 시장 ($B) | 2 | 8 | 20 |
| 위성 IoT ($B) | 3 | 7 | 15 |
| 이리듐 매출 ($M) | 831 | 950 | 1,200 |

### D2D (직접위성통신)

| 항목 | 내용 |
|------|------|
| 기술 | 스마트폰 직접 위성 연결 |
| 파트너 | 퀄컴 등 |
| 응용 | 긴급·SMS·데이터 (지상망 부재) |
| 경쟁 | 스타링크(SpaceX) + 애플(글로벌스타) |
| 이리듐 | LEO 66 위성 글로벌 커버리지 |

**D2D는 스마트폰 위성 연결 대형 신시장** — 이리듐 글로벌 커버리지 활용.

### 위성 IoT + 국방

| 사업 | 내용 |
|------|------|
| 위성 IoT | 자산 추적·M2M (저비용·저전력) |
| 국방(DoD) | 보안 위성통신 장기 계약 |
| 해상·항공 | 글로벌 커버리지 필수 |

### 시장의 특수성

| 구분 | 일반 통신 | 위성통신 |
|------|---------|------|
| 진입 장벽 | 보통 | 극단 (위성 CapEx) |
| 커버리지 | 지역 | 글로벌 (극지방) |
| 수익성 | 다양 | 인프라 후 고마진 |
| 부채 | 보통 | 높음 (위성 CapEx) |
| 변동성 | 보통 | 테마 변동 |

**핵심**: 위성통신은 위성 CapEx 진입 장벽 + 글로벌 커버리지. 인프라 구축 후 고마진 + FCF. D2D·IoT 성장. 부채 높음.
