2. 바이오텍 밸류에이션 방법론: rNPV 접근

왜 rNPV인가?

임상단계 바이오텍은 매출·이익이 없어 PER/EV/EBITDA 적용이 불가능하다. 업계 표준은 Risk-adjusted Net Present Value(rNPV)로, 각 파이프라인의 성공 시 기대 현금흐름에 임상 단계별 성공확률(PoS)을 적용하여 현재가치를 산출한다.

rNPV 산출 공식

rNPV = Σ (기대 현금흐름t × 누적 성공확률) / (1 + r)^t

임상 단계별 성공확률 (업계 평균)

단계 성공확률 누적 승인 확률
Phase 1 → Phase 2 65% —
Phase 2 → Phase 3 33% —
Phase 3 → NDA 60% —
NDA → 승인 85% —
Phase 1 → 승인 — ~11%
Phase 2b 완료 → 승인 — ~30%
Phase 2a 완료 → 승인 — ~20%
전임상 → 승인 — ~5%

※ FDA 통계 기반 업계 평균. 치료 영역별로 편차 존재.

적용 할인율

  • 임상 초기 단계: 15~20%
  • 임상 후기 단계: 12~15%
  • 본 분석 적용: 15% (임상 2상 단계 중심 포트폴리오)
