# AP위성 — 컨텍 통합 시너지·26E 흑전 컨센·K우주산업 정량 검증

---

## Executive Summary

AP위성 주가 +70% 폭등 (10,630→18,020원, 6개월) + 2026-01 정점 21,250원 후 박스권 (15,000~19,000원). 1·3월 정점 매물 흡수 후 회복. 핵심 동력은 (1) **컨텍(주) 최대주주 인수 = 우주 지상국·발사체·위성통신 통합 시너지**, (2) **2026E 컨센 매출 650억(+36%)·OP 80억(흑전)·ROE 6.55% 정상화**, (3) **K우주산업 정부 예산 +20%/년 + 누리호 4호기 (2026) + 차세대 발사체 (2032)** 가속, (4) **25Q4 매출 159억·OP +7억 분기 흑전 시그널**의 4중 회복 모멘텀. 2011년 설립 위성통신단말기 전문 + UAE 두바이 위성휴대폰 공급 + Baseband Modem SOC 기술. **무차입 + 자본 1,039억 + Cash Runway 23.4분기 + CB/BW·유증 0건**의 우량 자본구조. 다만 (1) **2025 매출 -18.5%·OP -43억 적전** 사이클 둔화 + (2) **분기 매출 50~270억 5배 변동성** + (3) **외인 -12.16만주 (20일 이탈)**의 단기 부담. **확률가중 PBR 기반 적정주가 21,766원 (현재 18,020원, +20.8% 업사이드)**. 박스권 상단 (19,000원) 거래량 동반 돌파 + 컨텍 시너지 매출 가시화 시 26,000~33,485원 (+44~86%) 가능.

## 1. 분석 배경

AP위성은 2025-11 10,630원 저점에서 2026-01 21,250원 정점 → 2026-02 15,000원 조정 → 2026-03~05 박스권 (15,000~19,000원) 형성. 핵심 동력:

### 동력 트리거 분석

**컨텍(주) 최대주주 인수**:
- 2024~ 우주 지상국·발사체·데이터 분석 전문 (주)컨텍 인수
- AP위성 (다운스트림·위성통신) + 컨텍 (업스트림·지상국·발사체) = 통합 시너지
- K우주산업 통합 플레이어 등장

**26E 흑전 컨센**:
- FnGuide 컨센 매출 650억 (+36% YoY)
- OP 80억 (흑전, OPM 12.3%)
- NI 70억, ROE 6.55% 정상화
- 25Q4 OP +7억 분기 흑전과 정합

**K우주산업 정부 정책**:
- 정부 우주 R&D 예산 +20%/년 (2026~2030)
- 누리호 4호기 (2026) + 차세대 발사체 (2032)
- 민간 위성 사업자 확대 (한화시스템·KAI·이노스페이스)

**무차입 + Cash Runway 23.4분기**:
- 이자부 부채 0
- 자본 1,039억, 현금 162억
- CB/BW·유증·스옵 0건

### 분석 범위

- 컨텍 통합 시너지의 정량 추정 (2026~2030)
- 26E 흑전 컨센의 추세 검증
- K우주산업 정부 예산 + 누리호·차세대 발사체 수혜 영향
- PBR 2.42x 옵션 가치 적정성
- 외인 이탈 + 분기 매출 변동성 단기 부담

### 한계

- 컨텍 인수 후 시너지 정량 데이터 미공개 (NDA)
- 분기 매출 5배 변동성 = 추세 검증 어려움
- FnGuide 4건 활발하나 정식 TP 별도 PDF 확인 필요
- K우주산업 정부 예산 정책 변동 위험
