### 핵심 사업: 자동차 샤시부품 전문사 (화신그룹 계열, 알루미늄 로어암·EV 신사업)

**2010년 설립, 화신(010690) 그룹 계열 자동차 샤시부품 전문사**. 최대주주 글로벌오토트레이딩(11.31%)은 화신과 동일 — 화신그룹 형제사. 샤시부품(로어암·서스펜션·언더바디) + 정밀가공 + 보수용 부품을 국내 완성차·계열사·1차 협력업체에 OEM 공급.

- **국내**: 경북 영천 본사 — 현대차·기아 + 화신 등 1차 협력업체 향 샤시부품 공급
- **해외**: LOCAL수출 + 직수출 (신용장). 화신그룹 글로벌 거점 연계
- **신사업**: ① **알루미늄 로어암 개발** (경량화 — EV 핵심 부품) ② 전기차 신사업 체제 구축 ③ 연구소 기술컨소시엄 — 미래차 시장 경쟁력 확보

### 비즈니스 특성

- **2026Q1 폭발적 턴어라운드**: 2024년 OP 13억(부진) → 2025년 118억(+778.7%) → **2026Q1 OP 122억(+557.7% YoY)** = 단분기에 2025 연간 OP 초과. V자 회복 진행
- **틈새시장 차별화**: 보수용부품·정밀가공·원소재 1차 가공으로 동종업체 차별화. TS16949 인증
- **경량화(알루미늄) + EV 전환**: 알루미늄 로어암은 EV 경량화 핵심 부품. ICE 강재 대비 단가·마진 우위
- **저PBR + 고배당**: PBR 0.72x + 배당수익률 5.6% + 화신그룹 시너지

### 고객 구조 & 원가 구조

- **고객 구조**: B2B 100% OEM. 국내 완성차(현대차·기아) + 계열사(화신) + 1차 협력업체. 45일 결제조건
- **매입/원가**: 강재·알루미늄(샤시 원소재) + 코일. 원소재 가격 + 환율 + 완성차 물량이 마진 변수. 원소재가공(COIL) 비중 31.4%로 원자재 직결

## 4. 성장 동력 분석

**동력 1: 2026Q1 OP 122억 폭발적 턴어라운드 ⭐⭐⭐⭐⭐ (◉)**
2024년 OP 13억(OPM 0.4%) 바닥 → 2025년 118억 → **2026Q1 OP 122억(+557.7%, OPM 10.1%)** = 단분기에 연간 OP 초과. 매출 +37.3% + 마진 극적 개선. **연환산 시 2026 OP 320억+ 가능** → PER 2.8x로 급락. 가장 강력한 동력.

**동력 2: 알루미늄 로어암 — EV 경량화 핵심 부품 ⭐⭐⭐⭐ (◉)**
**알루미늄 로어암 개발 완료 + 양산 진입**. 기존 강재 로어암 대비 경량화(EV 주행거리 핵심) + 단가·마진 우위. EV 1대당 알루미늄 부품 채택 증가 → 고부가 매출 믹스. 현대차그룹 EV 200만대 + 경량화 트렌드 직접 수혜. 마진 개선의 핵심 원천.

**동력 3: 화신그룹 시너지 + 현대차 물량 회복 ⭐⭐⭐ (◎)**
화신(010690)과 동일 최대주주(글로벌오토트레이딩). **화신그룹 글로벌 거점 + 메타플랜트 + 인도·미국 공급망 연계**. 현대차 신차 사이클(2026\~27) + 메타플랜트 가동률 회복 시 샤시부품 물량 동반 회복. 그룹 차원 수주·구매 시너지.

**동력 4: 외국인 +756,050주 폭매수 + 저PBR 가치주 ⭐⭐⭐ (◎)**
외국인 20일 +756,050주 폭매수(시총의 2.2%) + 기관 +12,670주. 1년 +73% (1,237→2,140) + 52주 신고가. PBR 0.72x + 배당 5.6%. **턴어라운드 폭발 + 외국인 매수가 결합한 모멘텀**. 다만 RSI 81 과매수.
