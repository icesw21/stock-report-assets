# 화신정공 T4 Deep Dive — 2026Q1 OP 폭발 턴어라운드 + 알루미늄 로어암 EV 경량화 (화신그룹 계열 샤시부품사 PBR 0.72x)

---

## Executive Summary

화신정공(126640)은 **화신(010690) 그룹 계열 자동차 샤시부품 전문사**다. 샤시류(로어암·서스펜션·언더바디, 매출 40.6%) + 원소재가공(31.4%) + 정밀가공·보수용 부품. 최대주주 글로벌오토트레이딩(11.31%)이 화신과 동일한 화신그룹 형제사.

핵심 투자 논리는 **3중 결합**: ① **2026Q1 OP 122억(+557.7%, OPM 10.1%) 폭발적 턴어라운드** — 단분기에 2025 연간 OP(118억) 초과 ② **알루미늄 로어암 — EV 경량화 핵심 부품** — 강재 대비 단가·마진 우위 ③ **PBR 0.72x + 배당 5.6% + 외국인 +756k 폭매수**.

2024년 OP 13억(OPM 0.4%) 바닥 → 2025년 118억(+778.7%) → 2026Q1 122억으로 극적 V자. PER 29.72x는 2025 저EPS(72원) 착시이며, **2026 OP 320억 추정 시 EPS \~770원 → PER 2.8x로 급락**. 1년 +73% 폭등 + 52주 신고가.

리스크는 **현대차·기아 단일 고객 의존 + 턴어라운드 지속성 검증(2\~3분기) + 52주 신고가 RSI 81 극단 과매수 + 소형주 변동성(시총 750억)**. 다만 CB/BW 0건 + 배당 5.6%로 하방 일부 견고.

**확률가중 적정주가 3,230원** (현재 2,140원 **+50.9%**) — 보수 2,200원(30%) / 기본 3,300원(50%) / 낙관 4,600원(20%). 전략: **MA20 1,789원 ~ MA60 1,621원 분할 매수, SL 1,500원, 목표 3,300원**.

## 1. 분석 배경

화신정공 주가는 2025년 저점 1,237원 → 2026년 2,140원으로 **1년 +73% 폭등**, 52주 신고가. 2026Q1 OP 122억(+557.7%) 폭발적 실적 + 외국인 +756,050주 폭매수가 랠리 동력. 2024년 OP 13억 바닥에서 극적 턴어라운드.

T1~T3에서 확인한 사실:
- 2026Q1 OP 122억(+557.7%, OPM 10.1%) — 단분기 연간 OP 초과
- 2024 OP 13억 → 2025 118억 → 2026Q1 122억 V자

**왜 이 주제가 핵심인가?** — 자동차 부품 턴어라운드주는 ① 실적 변곡점의 지속성 ② 신제품(알루미늄 로어암) 단가 믹스 ③ 밸류에이션 정상화의 3가지가 결합해야 리레이팅한다. 화신정공은 2026Q1 OP 폭발로 변곡점을 통과했고, 알루미늄 로어암이 마진 개선의 구조적 원천이며, PER 29.7x → 2.8x 착시 해소가 리레이팅 여지를 만든다. **턴어라운드 지속성 검증(2026Q2)이 핵심 분기점**인 모멘텀+가치 결합 베팅.

## 2. 산업/시장 개요

### 자동차 경량화 + EV 샤시부품 시장

**EV 전환의 핵심 과제 = 경량화**. 배터리 무게 증가로 차체 경량화가 EV 주행거리의 핵심. 강재 → 알루미늄 전환이 샤시(로어암·서브프레임)에서 가속. 알루미늄 로어암은 강재 대비 30\~40% 경량 + 단가 2\~3배.

| 구분 | 강재 로어암 | 알루미늄 로어암 |
|------|----------:|--------------:|
| 무게 | 기준 | -30\~40% |
| 단가 | 기준 | +100\~200% |
| 마진 | 보통 | 우위 (고부가) |
| EV 채택 | 감소 | 증가 |

### 현대차그룹 경량화 + EV 일정

| 시점 | 이벤트 | 화신정공 영향 |
|------|------|------|
| 2024 | 현대차 물량 둔화 | 매출 -10.5%, OP 13억 바닥 |
| 2025 | 물량 회복 + 알루미늄 로어암 | OP 118억 V자 |
| 2026 | 신차 사이클 + 경량화 가속 | OP 폭발 (1Q 122억) |
| 2027\~30 | EV 200만대 + 알루미늄 확대 | 단가 믹스 + 물량 |

### 시장의 특수성

| 구분 | 일반 부품 | 자동차 샤시부품 |
|------|---------|-----------|
| 진입 장벽 | 보통 | **높음 (안전·구조 검증)** |
| 고객 락인 | 약함 | **강함 (5\~7년 모델)** |
| 경량화 수혜 | 간접 | **직접 (알루미늄 전환)** |
| 사이클 | 자체 | **완성차 물량 동조** |
| 마진 변동성 | 보통 | **큼 (가동률 레버리지)** |

**핵심**: 샤시부품은 가동률 레버리지가 커서 물량 회복 시 마진 급증(2026Q1 OPM 0.4%→10.1% 입증). 알루미늄 로어암으로 단가 믹스 + 경량화 수혜 직접 수취.
