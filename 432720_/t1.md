### 핵심 사업: 초고속 인터페이스 반도체 IP 라이센싱

2017년 설립된 **반도체 IP 전문기업**으로 PCIe, UCIe, HBM 등 **초고속 인터커넥트 인터페이스 IP**를 개발하여 SoC 개발사 및 디자인하우스에 라이센싱 방식으로 공급한다. 2023년 코스닥 기술성장기업 특례상장 이후 최첨단 반도체 공정(3nm·5nm 등) 대응 IP 포트폴리오를 지속 확장하고 있다.

- **국내**: 삼성전자·SK하이닉스 생태계 내 SoC 개발사·디자인하우스 대상 IP 라이센싱
- **해외**: 글로벌 반도체 팹리스·디자인하우스 대상 PCIe/UCIe IP 공급 추진
- **신사업**: UCIe(Universal Chiplet Interconnect Express) IP — AI 가속기·HBM 기반 칩렛 아키텍처 수요 확대 시 핵심 수혜 포지션

### 비즈니스 특성

- **고마진 IP 라이센싱 모델**: 초기 개발비용 투자 후 반복 수익(royalty) 구조. 매출은 작으나 성공 시 레버리지 높음
- **높은 기술 집약도·낮은 변동비**: 인력 중심 R&D 구조로 일단 IP 개발 완료 시 추가 원가 거의 발생하지 않음. 단, 고정비(인건비·R&D) 부담이 절대적
- **공정 세대 전환 의존성**: 반도체 선단 공정(TSMC 3nm·TSMC 5nm 등) 전환 사이클에 따라 IP 채택 수요가 집중됨. 특정 공정 세대 전환 지연 시 매출 공백 발생

### 고객 구조 & 원가 구조

- **고객 구조**: B2B, SoC 개발사·디자인하우스 대상. 소수 고객 집중도 높을 가능성 (IP 업종 특성). 특정 고객 비중 공시 미확인
- **매입/원가**: EDA 툴 라이센스·인건비·R&D 비용이 핵심 원가. 반도체 IP는 무형자산 성격으로 제조 원재료 없음. 2025년 CAPEX 3.2억원으로 전년(16.1억원) 대비 대폭 축소 — 물리적 설비 투자 최소화

## 4. 성장 동력 분석

**동력 1: PCIe Gen5/6 및 UCIe IP 수요 확대 ⭐⭐⭐ (◉)**
AI 가속기(GPU·NPU)와 HBM을 연결하는 고속 인터커넥트 수요가 폭발적으로 성장하고 있다. 퀄리타스반도체는 PCIe Gen5/6, UCIe 등 최신 인터페이스 IP를 최첨단 공정(TSMC 5nm·3nm)에서 구현하는 기술을 보유하며, AI 반도체 설계 붐이 지속될수록 직접적인 IP 채택 수요 증가가 기대된다. 단일 IP 라이센싱 계약 체결 시 수억~수십억 원 규모의 매출이 일시 인식된다.

**동력 2: 칩렛(Chiplet) 아키텍처 확산에 따른 UCIe IP 특수 ⭐⭐⭐ (◎)**
반도체 산업이 단일 다이(monolithic)에서 칩렛(chiplet) 기반 패키지로 전환되면서 칩 간 고속 연결을 위한 UCIe 표준 IP 수요가 급증하고 있다. 퀄리타스반도체는 UCIe IP 상용화 역량을 보유하고 있으며, 국내외 AI·HPC SoC 개발사의 칩렛 설계 증가 시 핵심 IP 공급자로 포지셔닝 가능하다.

**동력 3: IP Sub-Licensing 수익 다각화 ⭐⭐ (◎)**
IP Sub-Licensing(매출 비중 20.15%)은 1차 라이센싱 이후 추가 수익을 창출하는 구조로, 동일 IP의 반복 활용 범위가 넓어질수록 고정비 부담 없이 매출을 늘릴 수 있다. 고객사의 설계 반복 사용(re-use) 수요가 증가할수록 수익성 개선 레버리지가 생긴다.

**동력 4: 해외 고객 확보를 통한 시장 확대 ⭐⭐ (○)**
현재 국내 SoC 개발사 및 디자인하우스 중심 영업 구조에서 글로벌 팹리스·OSAT 고객으로 확장 시 매출 기반이 크게 확대될 수 있다. 단, 해외 IP 시장은 ARM·Synopsys·Cadence 등 글로벌 대형 IP 벤더와의 경쟁 강도가 높아 진입 장벽이 존재한다.

## 8. Bull vs Bear

### Bull Case — "AI·칩렛 IP 수요 폭발, 선단 공정 IP 독과점"

◉ **PCIe Gen6·UCIe IP 대형 계약 체결** — AI 가속기·HBM 칩렛 설계 확산으로 고속 인터커넥트 IP 수요 급증. 단 1~2건의 대형 IP 라이센싱 계약으로 분기 매출이 수십억 원 단위로 점프 가능
◉ **최첨단 공정 진입 장벽** — TSMC 3nm·2nm 공정 대응 PCIe/UCIe IP는 개발 난도가 극히 높아 진입 장벽이 높음. 선행 개발 완료 시 선점 효과 지속
◎ **특관인 합계 42.23% 지분** — 최대주주 그룹의 높은 지분율이 경영 안정성 제공. 무분별한 희석 억제 유인
◎ **2025Q1 매출 +153% YoY** — 매출 회복 가시화의 초기 신호. IP 계약 파이프라인 개선 시 분기 매출 구조적 반등 가능
○ **외국인·기관 동반 순매수** — 최근 20일간 외국인 849,549주·기관 37,126주 순매수. 스마트머니 유입 초기 국면

### Bear Case — "끝나지 않는 적자, CB 오버행이 기다린다"

◉ **3년 연속 대규모 영업손실** — OPM -103% → -374% → -457%로 적자 폭 확대. IP 수익화 가시적 시점 없이 적자 구조 지속
◉ **2026년 9월 CB 전환권 행사 리스크** — 300억원 CB(전환가 12,942원) 보유자의 전환 인센티브(현재가 대비 +59% 업사이드)가 매우 높아 2026년 9월 이후 14.13% 추가 희석 확실시
◎ **Cash Runway 2.2분기** — 현 소진 속도 유지 시 2026년 2분기 내 추가 자금 조달 불가피. 추가 유상증자 또는 CB 발행 시 주주 가치 추가 희석
◎ **매출 극소수 고객 의존** — 분기 매출 편차 극대화(4.4억~37.2억). IP 계약 1건 체결 지연 시 분기 매출 제로 수준으로 급감
○ **현재가 상한가 수준 → 기술적 과열** — 52주 저가(10,190원) 대비 +102% 상승 후 상한가. 단기 차익실현 매물 압력 집중 가능
