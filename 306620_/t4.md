# 지아이에스 — 반도체·MLCC Dicing 본업 + 푸드테크 보글봇 옵션 vs BW 13.41% 인-더-머니 희석·기관 -155만주 이탈 정량 검증

## Executive Summary

지아이에스(306620)는 2000년 설립·2018년 분할 신설된 KOSDAQ **반도체·MLCC 절단장비(Dicing·Singulator·MLCC Cutter) + 카메라 모듈 세정 + AI 로봇 시스템** 종합 특수기계 기업이다. 매출 구성은 자체 제조 제품 85.36% + 상품 14.44% + 기타 0.20%로, 본업은 글로벌 DISCO(점유율 80%+)·Accretech가 과점한 절단장비 시장에서 한국 niche 포지션을 점한다. 신사업으로 **푸드테크 자동조리 로봇 '보글봇'**을 개발 완료했으며, AI 로봇 시스템과 결합한 옵션 가치를 보유한다. 현재가 **3,165원 / 시총 1,517억 / PBR 3.09x / 적자 / ROE -8.5% / 부채비율 333.2%**.

5년 추세는 매출 +52% 성장(499→758억)이나 단조롭지 않다. **2024년 매출 1,045억(+20.4%) + NI 흑전 (+2.6억)** → **2025년 매출 758억(-27.4%) + NI 재차 적자 (-36억)** = 흑전 정착 실패. 분기 변동성이 극심하여 24Q1 매출 454억(+256.6%, 일회성 대형 수주 추정) → 24Q3~Q4 170~177억 급락 → **25Q1 매출 76억 분기 바닥(-83.3% YoY)** → 25Q2~Q4 211~240억 회복 → 25Q4 OP 25억(+558%) 회복 → **26Q1 매출 139억(+82.9% YoY) but OP -14억 재차 적자**. 분기 매출이 76~454억 사이를 넘나드는 수주 리드타임 의존형 사이클.
