## 2. 사업 구조

### 사업 개요

**칩모스는 대만 OSAT(반도체 후공정 - 패키징·테스트) 중견 기업**(미국 ADR 상장)이다. 5개 세그먼트: **Testing(테스트) + Assembly(어셈블리) + DDIC(디스플레이 드라이버 IC) 패키징·테스트 + Bumping(범핑) + 기타**. DDIC 후공정 + 메모리(NOR/NAND) 테스트 특화. 대만·일본·중국 생산.

### 사업부문별 매출 구조 (FY2025 추정)

| 사업부문 | 주요 서비스 | 비중(%) | 비고 |
|---------|---------|------:|-----|
| DDIC 패키징·테스트 | 디스플레이 드라이버 IC | \~40 | 핵심, COG/COF |
| Testing | 메모리·로직 테스트 | \~30 | NOR/NAND·ASIC |
| Assembly | 리드프레임·기판 패키지 | \~20 | FBGA·QFP |
| Bumping·기타 | 범핑·기타 | \~10 | |

※ FY2025 추정. DDIC + 메모리 테스트 특화. 단위: 세그먼트 추정 비중.

### 핵심 사업: 대만 OSAT (DDIC·메모리 후공정 + 고배당)

**반도체 후공정(패키징·테스트) 전문**. DDIC(디스플레이 드라이버 IC) 후공정 글로벌 선두권 + 메모리(NOR/NAND) 테스트. ASE·Amkor 등 대형 OSAT 대비 중견이나 DDIC·메모리 특화. **고배당 정책**(배당 이력 2022 $2.92 등).

- **DDIC 패키징·테스트**: 디스플레이 드라이버 IC COG(Chip-on-Glass)/COF(Chip-on-Film). OLED·LCD 패널
- **Testing**: 메모리(NOR/NAND) + 로직·ASIC 테스트. 고속 디지털 테스트
- **Assembly·Bumping**: 리드프레임·기판 패키지 + 범핑

### 비즈니스 특성

- **반도체 후공정 사이클 회복**: 메모리·디스플레이 다운사이클 → 회복. PER 83.7x는 사이클 저EPS 착시. 사이클 회복 시 EPS 정상화
- **DDIC·메모리 특화**: DDIC 후공정 선두권 + 메모리 테스트. OLED 패널 + 메모리 수요 수혜. AI·디스플레이 사이클
- **고배당 정책**: 배당 이력 풍부(2022 $2.92, 2021 $1.57). 대만 기업 고배당 성향. 안정 주주환원
- **1년 +344% 급등 + 과매수**: $15.09→$70.26. 반도체 후공정 회복 + AI 수요. RSI 75.8 과매수

### 고객 구조 & 원가 구조

- **고객 구조**: 팹리스·IDM (DDIC·메모리·로직). 대만·중국·일본 반도체사. B2B 후공정 위탁
- **매입/원가**: 후공정 장비·소재 + 인건비. 가동률 + 후공정 단가 + 사이클이 마진 변수. 흑자(OP margin 7.5%)

## 4. 성장 동력 분석

**동력 1: 반도체 후공정 사이클 회복 ⭐⭐⭐⭐ (◉)**
메모리·디스플레이 다운사이클(FY2023\~24) → **회복 사이클**. AI 서버·메모리 + OLED 디스플레이 수요 회복. 후공정 가동률 상승 + 단가 개선. PER 83.7x 저EPS → 사이클 회복 시 EPS 정상화. 가장 강력한 동력.

**동력 2: DDIC 후공정 + OLED 디스플레이 ⭐⭐⭐⭐ (◉)**
**DDIC(디스플레이 드라이버 IC) 후공정 글로벌 선두권**. OLED 패널 확산(스마트폰·TV·차량) + DDIC 수요. COG/COF 패키징·테스트. 디스플레이 사이클 회복 수혜.

**동력 3: 메모리 테스트 (NOR/NAND) ⭐⭐⭐ (◎)**
메모리(NOR/NAND) 테스트 특화. AI·IoT·자동차 메모리 수요. 메모리 업사이클 시 테스트 물량 증가. 후공정 회복 동력.

**동력 4: 고배당 + 안정 주주환원 ⭐⭐⭐ (◎)**
대만 기업 고배당 성향. 배당 이력 풍부(2022 $2.92, 2021 $1.57). 사이클 회복 + 흑자 시 배당 증가. 안정 주주환원 + 밸류 매력.
