# 칩모스 T4 Deep Dive — 대만 OSAT 후공정 사이클 회복 + DDIC·메모리 (고배당 vs PER 83.7x·1년 +344% 과열)

---

## Executive Summary

칩모스(IMOS)는 **대만 OSAT(반도체 후공정 - 패키징·테스트) 중견 기업**(미국 ADR)으로, **DDIC(디스플레이 드라이버 IC) 후공정 글로벌 선두권 + 메모리(NOR/NAND) 테스트 특화**다. 5개 세그먼트(테스트·어셈블리·DDIC·범핑·기타).

핵심 투자 논리는 **후공정 사이클 회복 + DDIC·메모리 + 고배당**: ① **반도체 후공정 사이클 회복** — 메모리·디스플레이 다운사이클 종료, PER 83.7x 저EPS 정상화 ② **DDIC 후공정 + OLED 디스플레이** — 디스플레이 드라이버 선두권 ③ **고배당 정책** (2022 $2.92 등). 흑자(OP margin 7.5%).

1년 +344% 급등($15.09→$70.26)은 후공정 회복 + AI 수요. PER 83.7x는 사이클 저EPS($0.80) 착시로 FY2026E EPS $2.2 정상화 시 30x. 단 RSI 75.8 과매수 + ASE·Amkor 대비 높은 밸류.

**확률가중 적정주가 $58.4** (현재 $66.94 **-12.8%, 고평가**) — 보수 $45(40%) / 기본 $60(40%) / 낙관 $82(20%). 전략: **MA20 $55 ~ $51 눌림목 분할 매수, 후공정 회복·EPS 정상화 가시화 시 진입**.

## 2. 산업/시장 개요

### 반도체 후공정(OSAT) 사이클

**OSAT는 반도체 사이클 후행 + 회복 진입**. 메모리·디스플레이 다운사이클(FY2023\~24) → FY2025\~26 회복. AI 서버·메모리 + OLED 디스플레이 수요가 후공정 물량 견인.

| 구분 | FY2022 | FY2024 (저점) | FY2026E |
|------|----:|----:|-----:|
| 글로벌 OSAT 시장 ($B) | 45 | 40 | 52 |
| DDIC 후공정 수요 | 높음 | 둔화 | 회복 |
| 칩모스 매출 ($M) | 920 | 720 | 900 |
| 칩모스 EPS ($) | 3.5 | 0.8 | 2.2 |

### DDIC 후공정 + OLED

| 항목 | 내용 |
|------|------|
| DDIC | 디스플레이 드라이버 IC |
| 패키징 | COG(Chip-on-Glass)/COF(Chip-on-Film) |
| 응용 | OLED·LCD 패널 (스마트폰·TV·차량) |
| 칩모스 | DDIC 후공정 글로벌 선두권 |

**OLED 확산(스마트폰·TV·차량) → DDIC 후공정 수요** — 칩모스 핵심.

### 메모리 테스트

| 항목 | 내용 |
|------|------|
| 메모리 | NOR/NAND 테스트 |
| 응용 | AI·IoT·자동차 |
| 사이클 | 메모리 업사이클 수혜 |

### 시장의 특수성

| 구분 | 일반 반도체 | OSAT 후공정 |
|------|---------|-----------|
| 사이클 | 보통 | 후행 + 변동 |
| 수익성 | 다양 | 가동률 의존 |
| 배당 | 보통 | 고배당 (대만) |
| 밸류 | PER | 사이클 PER + EV/EBITDA |
| 변동성 | 보통 | 사이클 변동 |

**핵심**: OSAT는 반도체 후공정 사이클 + 가동률 의존. DDIC·메모리 특화 + 고배당. 사이클 회복이 EPS·밸류 관건.

## 9. 결론

칩모스는 **대만 OSAT(반도체 후공정) 중견 + DDIC·메모리 테스트 특화 + 고배당**으로, **반도체 후공정 사이클 회복 + OLED 디스플레이 + 메모리 수요**가 동력이다. DDIC 후공정 글로벌 선두권 + 메모리 테스트. 흑자(OP margin 7.5%) + 고배당. PER 83.7x는 사이클 저EPS($0.80) 착시로 FY2026E EPS $2.2 정상화 시 30x.

1년 +344% 급등은 후공정 회복 + AI 기대. 단 RSI 75.8 과매수 + ASE(18x)·Amkor(16x) 대비 높은 밸류(정상화해도 30x). 확률가중 적정주가 $58.4는 현재가 $66.94 대비 -12.8% 고평가.

**핵심 트레이딩 시그널**: ① MA20 $55 ~ $51 눌림목 분할 매수 ② 후공정 사이클 회복·EPS 정상화 ③ DDIC·메모리 수요 ④ 신고가권 추격 비추.

### 모니터링 체크포인트

- ✅ **후공정 사이클 회복** — 메모리·디스플레이 가동률·단가 (EPS 정상화 핵심)
- ✅ **DDIC·OLED 수요** — 디스플레이 드라이버 후공정
- ✅ **메모리 테스트 물량** — NOR/NAND 업사이클
- ✅ **OP margin 회복** — 7.5% → 11%+ (FY2026E)
- ✅ **PER 83.7x + 1년 +344%** — 사이클 저EPS 착시 + 과매수 변동
- ✅ **배당** — 고배당 정책 유지·증가

### 투자 결론

**HOLD → 눌림목 분할 매수 (OSAT 사이클주)** — 반도체 후공정 사이클 회복 + DDIC·메모리 + 고배당 펀더멘털은 견고하나, PER 83.7x(사이클 저EPS 착시) + 1년 +344% + RSI 75.8 과매수로 단기 과열. 현재가 $66.94는 적정가($58.4) 대비 -12.8% 고평가. **MA20 $55 ~ $51 눌림목 분할 매수** 권고. 12\~24개월 시계에서 ① 후공정 사이클 회복 + ② DDIC·메모리 성장 + ③ EPS 정상화 결합 시 **기본 $60 ~ 낙관 $82(+23%) 진입 여지**.

오늘 함께 분석하는 BAND·IRDM과 비교 시, 칩모스는 흑자 + 고배당 OSAT로 펀더멘털 안정적이나 PER 83.7x 고평가가 부담. 포트 2\~4% 한도. **후공정 사이클 회복·EPS 정상화가 밸류 정당화 핵심 분기점** — 가시화 시 정상화, EPS 미달 시 -20\~30% 조정. 고배당 수취하며 사이클 회복 대기 + 대만 지정학·환율 유의.
