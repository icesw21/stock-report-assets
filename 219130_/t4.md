# 타이거일렉 — AI·HBM 반도체 검사 PCB 슈퍼사이클 + 26Q1 OPM 13.1% 마진 폭발 + 티에스이 그룹 시너지 + 5.75배 폭등 후 극단 과열 정량 검증

## Executive Summary

타이거일렉(219130)은 2000년 설립·2011년 ㈜티에스이(131290) 자회사로 편입된 KOSDAQ **반도체 후공정 검사용 고다층·고밀도 PCB 전문 제조사**다. 매출 구성은 (1) **Probe Card용 PCB 50.51%** — 반도체 검사 핵심 (HBM·D램·AI 칩 직수혜), (2) **Load Board용 PCB 26.51%** — 패키지 테스트, (3) **일반 Board용 PCB 13.03%**, (4) **Socket Board용 PCB 9.40%**로, 후공정 검사 PCB에 특화된 niche 강자다. 모회사 ㈜티에스이(테스트 핸들러) + 메가터치(프로브 카드 본체) + 타이거일렉(검사 PCB) = **티에스이 그룹 반도체 후공정 통합 라인업**. 현재가 **67,800원 / 시총 4,281억 / PER 1,540.91x / PBR 8.08x / ROE 9.0% / 부채비율 154.9%**로 52주 신고가 (11,800원→67,800원, **5.75배 폭등**).

폭등 동력은 (1) **2026Q1 매출 268억(+57.7% YoY) + OP 35억(+250.9%) + NI 28억(+256.4%) + OPM 13.1% 정점급 마진** — 분기 매출 132억(24Q1) → 268억으로 2년간 +103% 가속, (2) **2023~2024 적자(-49억 누적 NI) → 2025 V자 폭등** (OP +476.6%, NI +1817.4%) — 본업 정상 마진 도달, (3) **AI 반도체·HBM 슈퍼사이클** — SK하이닉스·삼성전자·마이크론 HBM3E·HBM4 capex 폭증 + B200·MI300·TPU 후공정 검사 PCB 수요 폭증, (4) **㈜티에스이 43.71% + 이경섭 8.41% = 52.12% 안정 지배구조** + 모회사 그룹 시너지, (5) **CB/BW·유상증자 0건** = KOSDAQ에서 매우 드문 깨끗한 오버행, (6) **외인 +75,845 + 기관 +32,722 20일 누적 동시 매집** + 유동비율 218.2% 견조한 재무.

리스크는 (1) **52주 100% 상단(5.75배 폭등) + RSI 77.8 + MA200 +141.3% 매우 극단 과열** = 단기 평균회귀 압력 매우 큼, (2) **PBR 8.08x · PER 1,540.91x(TTM EPS 44원 왜곡)** = 정점급 멀티플, Fwd PER 38.9x로 환산해도 KOSDAQ 평균 대비 프리미엄, (3) **26Q1 단일 분기 가속의 추세 확정 미확인** — 2~3분기 OPM 10%+ 연속 유지 검증 필요, (4) **모회사 의존도** — 티에스이 그룹 수주 변동 시 매출 직접 영향, (5) **HBM 사이클 변동성** — 메모리 사이클 둔화 시 검사 PCB 단가·물량 동반 감소.

확률가중 적정주가 **64,168원 (-5.4%)** — 현재가와 거의 일치하는 적정 영역. **26Q2~Q4 OPM 10%+ 연속 유지 + 매출 280억+ 안정화** 시 PBR 10x (+24%, 83,880원) 진입 가능. **한미반도체(PBR 45.69x) 정상화 비교** 시 PBR 15x (+86%, 125,820원) 극낙관 시나리오 가능 (확률 5%). 단기는 RSI 77.8 + MA200 +141.3% 매우 극단 과열로 **-20~-30% 평균회귀 우선 가정**.

**전략**: 보유자 30~50% 익절 + 잔량 홀드. 신규 진입 1차 **54,965원(MA20)** 30% / 2차 **45,000원(추세선)** 30% / 3차 **39,661원(MA60)** 30%. SL 28,094원 (MA200, BPS 8,388의 +235% 마진). 1년 목표 80,000원 (+18%), 2년 100,000원 (+47%). AI 반도체·HBM 슈퍼사이클 직수혜 + 티에스이 그룹 시너지가 결합된 KOSDAQ 우량 종목이나, 단기 과열은 극단적.

### 1.2 핵심 기술 — 반도체 후공정 검사 PCB

반도체 검사 공정용 PCB는 **고다층(12~20층)·고밀도(100μm 미만)·고온 내구성**이 핵심 요구 사양으로, 일반 PCB 대비 단가 5~10배. HBM(High Bandwidth Memory) 검사용 PCB는 **TSV(Through-Silicon Via)·micro-bump 검증** 필요로 일반 D램 대비 2~3배 단가.

#### Probe Card PCB
- 웨이퍼 단계 검사 (KGD: Known Good Die)
- HBM3E (12층 stack), HBM4 (16층 stack) 검사 단가 폭등
- 글로벌 경쟁: Ibiden(일본), Unimicron(대만), Zhen Ding(대만), ISU페트라시스(한국)

#### Load Board PCB
- 패키지 후 최종 테스트 단계
- 글로벌 ATE(Automated Test Equipment) 메이커 Advantest·Teradyne 향 OEM
- AI 칩(NVIDIA B200, AMD MI300, Google TPU) 후공정 검사 폭증

그룹 통합으로 **반도체 후공정 토탈 솔루션** 제공 가능 → 글로벌 OEM(SK하이닉스·삼성전자·마이크론·TSMC) 향 단일 공급 채널.

**AI 반도체·HBM 슈퍼사이클** 직수혜. SK하이닉스 HBM3E/HBM4 capex 2024~2027 약 30조 + 삼성전자 HBM 양산 본격화 + 마이크론 HBM 경쟁 = 검사 PCB 수요 폭증. 글로벌 PCB 경쟁자 대비 후공정 검사 PCB 특화로 진입장벽 명확.
