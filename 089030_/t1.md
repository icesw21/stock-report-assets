## 2. 사업 구조

### 핵심 사업: 반도체 테스트 핸들러 + 큐브프로버(WLBI)

테크윙은 2002년 설립된 **반도체 검사장비 전문기업**으로, 메모리 **테스트 핸들러 글로벌 1위**다. SK하이닉스·삼성전자 등에 공급한다.

- **테스트 핸들러(본업)**: 메모리·SoC·SSD 테스트 시 디바이스를 핸들링하는 장비. 글로벌 점유율 1위, 안정적 매출 기반
- **COK(Change Over Kit)**: 핸들러 소모성 교체키트 — 반복 매출
- **큐브프로버(Cube Prober, 신성장)**: 웨이퍼레벨 번인(WLBI)·테스트 장비 — **HBM 등 첨단 메모리 테스트** 핵심. 최대 투자포인트
- **확장**: Memory Test Handler → SoC·SSD·Burn-in으로 영역 확대

### 비즈니스 특성

- **메모리 capex·테스트 수요 연동**: 본업은 메모리 투자·테스트 물량에 연동
- **큐브프로버 옵셔널리티**: HBM 생산 확대 시 WLBI 큐브프로버 신규 대형 시장
- **순이익 변동성**: 부채비율 298.7%로 이자비용·환손익이 순이익을 크게 흔듦 → 영업이익 기준 판단 필요

## 4. 성장 동력 분석

**동력 1: 큐브프로버(HBM WLBI) ⭐⭐⭐ (◉)**
웨이퍼레벨 번인·테스트 장비로 HBM 등 첨단 메모리 테스트 핵심. **공급 확대 가시화**(신한 평가). HBM 생산 확대 시 신규 대형 시장 — 테크윙 최대 투자포인트.

**동력 2: 메모리 테스트 핸들러 본업 ⭐⭐⭐ (◉)**
글로벌 점유율 1위 본업이 메모리 capex 사이클로 회복(2026Q1 매출 +51.5%). 안정적 실적 뒷받침.

**동력 3: COK 반복 매출 ⭐⭐ (◎)**
핸들러 소모성 교체키트(COK)로 설치 기반 반복 매출. 안정성.

**동력 4: SoC·SSD·Burn-in 확장 ⭐⭐ (○)**
메모리에서 SoC·SSD·번인으로 제품 영역 확대, 고객·시장 다변화.

## 8. Bull vs Bear

### 🐂 Bull Case — "큐브프로버(HBM WLBI) 신성장 + 본업 회복"

◉ **큐브프로버 공급 확대** — HBM WLBI 신규 대형 시장, 멀티플 상단↑
◉ **본업 회복** — 핸들러 1위, 메모리 capex로 2026Q1 매출 +51.5%·OP +444%
◎ **COK 반복 매출** — 설치 기반 안정성
◎ **SoC·SSD·번인 확장** — 제품 다변화
○ **외국인 순매수** — +52.7만주

### 🐻 Bear Case — "순이익 변동 + 부채 + 고평가"

◉ **순이익 변동·부채 298%** — 금융비용·환손익, PER 196x 착시
◉ **밸류 부담** — PBR 8.23x, 큐브프로버 선반영
◎ **큐브프로버 실행 리스크** — HBM 채택·물량 지연
◎ **메모리 capex 사이클** — 본업 투자 둔화 시 수주 감소
○ **수급·최대주주 지분** — 기관 매도, 지분 13.21%

## 9. 종합 판단

**기업의 체질**: 메모리 테스트 핸들러 글로벌 1위로, **큐브프로버(WLBI)라는 HBM 테스트 신성장축**을 보유한 반도체 검사장비사다. 2026Q1 매출 +51.5%·영업이익 +444%로 본업이 회복했고, 신한은 큐브프로버 공급 확대 가시화를 근거로 "2026 주목 소부장"으로 평가한다. 다만 **부채비율 298.7%로 이자비용·환손익이 순이익을 극심하게 흔들어**(2024 OP +234억에도 순손실 -220억) PER 196x는 저EPS 착시이며, 영업이익·매출 추세와 큐브프로버 성장으로 판단해야 한다. 1년 고점(71,100원) 대비 -30% 조정 국면.

**투자 가치가 있는 조건**:

| 조건 | 시점 | 확인 방법 |
|------|------|-----------|
| 큐브프로버 HBM 채택·물량 | 2026~ | 수주·고객(SK하이닉스 등) |
| 본업 핸들러 회복 | 분기별 | 매출·영업이익 |
| 순이익 정상화 | 분기별 | 금융비용·환손익 안정 |

**핵심 모니터링**:
- ✅ **큐브프로버(HBM WLBI) 공급·수주** — 최대 성장축
- ✅ **본업 핸들러 매출·영업이익** — 메모리 capex 회복 (2026Q1 +51.5%)
- ⚠️ **순이익 변동·부채비율 298%** — 금융비용·환손익, 영업이익 기준 판단
- ⚠️ **PER·PBR 밸류** — 큐브프로버 성장 선반영 vs 실현
