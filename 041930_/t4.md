# SY동아 — PBR 0.56x 극저평가 + ROE 14.5% 펀더멘털 가치 재발견의 정량 검증

---

## Executive Summary

SY동아 주가 2025-11 저점 5,620원에서 2026-05-19 7,200원까지 6개월간 **+28% 안정 상승** + 당일 박스권 돌파 +14%. 1974년 설립 KOSDAQ 자동차·가전 특수고무부품 글로벌 공급사. 핵심 동력은 (1) **PBR 0.56x + PER 4.13x = Peer 평균 대비 -46~-60% 명백한 저평가**, (2) **5년 매출 +40% 누적 성장 + 5년 연속 흑자 + 2025 ROE 14.5% 가속**, (3) **FCF +131억 + CB/BW·유증·스옵 0건의 깨끗한 자본구조**, (4) **외인 +8.2만주 + 기관 +1.6만주 (20일) 매집 + 임경식 40.65% 안정 지배**의 5중 가치 정합 시그널. 가전 50.6% + 자동차 49.4% 매출 균형 + **인도·중국·러시아 현지생산** 글로벌 헤지. 25Q1~Q3 매출 1,000억+ 분기 사상 최대 영역. 25Q4 OP 1억 일시 둔화 (계절성·환율). 확률가중 PBR 기반 **적정주가 10,983원 (현재 7,200원 대비 +52.5% 업사이드)**. 박스권 돌파 + RSI 65 건전한 추세 시작 — **단기 -5~-10% 조정 후 분할 매수 또는 즉시 진입 가능 영역**.

## 1. 분석 배경

SY동아는 KOSDAQ 자동차·가전 특수고무부품 업체로 **5년 연속 흑자 + 매출 +40% 누적 성장**의 안정 펀더멘털을 가졌으나 **PBR 0.56x**의 명백한 저평가 영역. 핵심 동력:

### 동력 트리거 분석

**저평가 + 펀더멘털 우위 정합**:
- PBR 0.56x = 자본의 56% 수준 (청산가치)
- PER 4.13x = Peer 평균 10.4x의 -60%
- ROE 14.5% = Peer 평균 약 8%의 +6.5%p 초과

**5년 누적 성장**:
- 매출 2,888억 (2021) → 4,034억 (2025) = +40%
- 5년 연속 OP 흑자 (140~220억대)
- 분기 매출 1,000억+ (25Q1~Q3) 안정

**외인·기관 동시 매집**:
- 외인 20일 +8.21만주
- 기관 20일 +1.63만주
- 시장 핵심 투자자 가치 재발견 시그널

### 분석 범위

- PBR 0.56x 저평가의 정당성·평균회귀 시나리오
- 5년 연속 흑자 + 매출 성장의 추세성
- 인도·중국·러시아 현지생산 글로벌 헤지 효과
- 박스권 돌파 후 가치 재발견 진행

### 한계

- FnGuide 정식 커버리지 부재 → 컨센 EPS 자체 추정
- 25Q4 OP 1억 일시 둔화 (분기 회복 26Q1 검증 필요)
