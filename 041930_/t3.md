## 6. 밸류에이션 논평

현재 7,200원은 **PBR 0.56x + PER 4.13x로 명백한 저평가 영역**. ROE 14.5% + 5년 연속 흑자 + FCF +131억의 펀더멘털과 가격 괴리. 확률가중 적정주가 **10,983원 (+52.5%)**.

**프리미엄/디스카운트 요인**:

| 요인 | 방향 | 조정폭 |
|---|---|---:|
| 5년 연속 흑자 + ROE 14.5% | ↑ | +20%p |
| FCF +131억 + CB/BW 0건 | ↑ | +15%p |
| 외인 +8.2만주·기관 +1.6만주 매집 | ↑ | +15%p |
| 박스권 돌파 + RSI 65 건전 | ↑ | +10%p |
| 임경식 40.65% 안정 지배 | ↑ | +5%p |
| 부채비율 163.7% (개선 중) | ↓ | -5%p |
| 사이즈 디스카운트 (1,094억) | ↓ | -10%p |
| 글로벌 자동차·가전 둔화 | ↓ | -10%p |
| FnGuide 커버리지 부재 | ↓ | -5%p |

**재평가 트리거**:
1. PBR 0.85x 도달 (10,888원, +51%)
2. 26Q1~Q4 분기 OP 50~60억 안정
3. 부채비율 150% 이하 감축
4. Peer 평균 평균회귀 (PBR 1.0x = 12,809원)

**결론**: **국내외 자동차·가전 부품 Peer 대비 명백한 저평가 + 펀더멘털 우위** — 확률가중 +52% 업사이드. 박스권 돌파 동반 진입 가능 영역. MA20 (6,702원) re-test 시 분할 매수 권장.
