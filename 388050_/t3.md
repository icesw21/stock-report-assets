## 6. 밸류에이션 논평

**핵심 판단**: 현재가 16,250원은 PBR 기본(13,512원)·PER Fwd 기본(18,000원)·PER TTM 낙관(17,820원)이 만나는 **기본~낙관 시나리오 중간 영역**에 위치. 신규 수주 모멘텀(한빛 3·4호기, 체코 두코바니, 액침냉각 ESS 첫 외부 수주, 미국 항만청 사업) 가시화 시 **20,000원대(낙관 PBR 9x = 20,268원, 낙관 블렌드 21,696원)** 진입 여력. 반대로 26Q1 비수기 적자 재현·신한울 매출 인식 지연·외국인 매도 가속 시 **14,000원대(기본 블렌드 14,464원, PBR 6x = 13,512원)** 회귀 압력. 현재 위치는 **상승여력 +25~33% vs 하락여력 -10~17%**, 리스크/리워드 비대칭이 약하게 우호.

**프리미엄/디스카운트 요인**:

| 요인 | 방향 | 조정폭 |
|------|------|--------|
| 코스닥 소형주 디스카운트 (시총 3,040억, 메이저 1/100) | ↓ | -20%p |
| 외국인 지분율 0.0% + 20일 누적 -2.6억 매도 | ↓ | -10%p |
| 분기 변동성 + Q4 단일분기 48% 집중 계절성 | ↓ | -8%p |
| 부채비율 213% 표면 시그널 (실질 무차입) | ↓ | -5%p |
| 원전 배전반(KEPIC·Q-Class) 진입장벽 프리미엄 | ↑ | +25%p |
| 액침냉각 ESS 양산(2026.4) + AI 데이터센터 옵션 | ↑ | +15%p |
| 관급 No.1 + 26 정부예산 +8.1% YoY 우호 매크로 | ↑ | +10%p |
| 26E OP +47.9% 레버리지 가시성 | ↑ | +12%p |
| **순(net) 조정** | ↑ | **+19%p** (premium 우위) |

**재평가 트리거**:
1. **한빛 3·4호기 / 체코 두코바니 입찰 결과 (1H26)** — 700억+ 수주 시 잔고 +60%, PER 멀티플 정당화
2. **액침냉각 ESS 첫 외부 수주 (2H26)** — 데이터센터·공공기관 향 첫 매출 인식, 옵션가치 → 본업 전환
3. **26Q1 실적 (2026.05) — 25Q1 적자 → 26Q1 흑전 + OPM 두 자릿수 유지** 시 분기 변동성 디스카운트 해소
4. **신한울 매출 인식 + 부채비율 200% 이하 정상화 (2026 연말)** — 외부 신용 평가 정상화

**결론**: 현재 16,250원은 **기본 시나리오를 넘어 낙관 시나리오의 일부를 선반영한 가격**. 단기적으로 1H26 입찰 결과·26Q1 실적이 멀티플 방향성을 결정하는 변곡점. **상승 여력은 한빛/체코 수주 등 외부 이벤트 의존도가 크고, 하락 여력은 13,500~14,500원의 PBR 6x · PER Fwd 기본선이 1차 지지대**. 기존 보유자는 홀드 + 신규 진입은 14,000원대 조정 시 분할 접근이 리스크-리워드 측면 합리적.
